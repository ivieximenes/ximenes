Defeitos

 

•Corrigido o defeito de concorrência no Chat;
•Corrigido o defeito de replicação para o cluster do XML de importação de usuário do Active Directory;
•Corrigido o defeito na associação de membros de grupos durante importação do Active Directory;
•Corrigido o defeito de falha no deploy de módulos quando não existe o diretório 'lumisdata/data/temp';
•Corrigido o defeito de vazamento de informação no LocalThread pelo ExecutorManager;
•Corrigido o defeito de acesso a arquivos estáticos de dentro de portlets;
•Corrigido o defeito de código de HTTP 404 ao acessar página em cache não gerada;
•Corrigido o defeito de cache de query no hibernate;
•Corrigido o defeito de replicação ao desinstalar módulos em cluster;
•Corrigido o defeito de execução de javascripts presentes no corpo HTML de instância de interface com renderização em modo client-side script;
•Corrigido o defeito de SQL com mais de 1000 itens durante a limpeza de cache HTML em ambiente Oracle;
•Corrigido o defeito de uso de caracteres especiais no controle de seleção de conteúdos hierárquicos;
•Corrigido o defeito de importação/exportação do atributo cachedpage ao importar um canal;
•Corrigido o defeito de limpeza excessiva de caches de memória quando ocorre um acesso a uma página com cache HTML ainda não gerada.

 

Melhorias

 

•Incluída a propriedade lumis.portal.url.preservePreviousParameters, que habilita o suporte para que URLs mantenham o estado anterior da página. Por padrão, o estado não será mantido.
