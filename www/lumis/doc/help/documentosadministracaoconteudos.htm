Administração do Conteúdo

O serviço de Documentos possui um ambiente administrativo disponibilizado pelo administrador do portal, para adição do documento no repositório, pelo editor responsável. Uma vez disponibilizado o documento no repositório, é possível re-editar suas propriedades, mudá-lo de diretório, ou excluí-lo do repositório.

Criação de Pastas

O repositório de documentos pode ser estruturado em pastas permitindo uma maior organização dos documentos adicionados ao repositório. Para criá-las, basta selecionar “Criar Pasta” e informar o nome. Inicialmente, as pastas são criadas abaixo da pasta raiz, representada por “/”.  Para criar pastas aninhadas, basta selecionar a pasta destino e novamente selecionar “Criar Pasta”.

As pastas são listadas na área administrativa para manutenções posteriores.

Arquivo e/ou pasta pode ser movido para outro diretório. Para isso, deve-se selecionar tal arquivo e/ou pasta, acionar "Mover", e escolher o diretório destino.

Adição de Documentos

Para disponibilizar um documento no repositório é necessário primeiramente selecionar o diretório destino, acionar "Adicionar" e preencher as seguintes informações:


 

•Nome do Arquivo: Arquivo que estará disponível no repositório;
•Título: Nome descritivo para o arquivo;
•Descrição: Texto explicativo sobre o documento.

Edição de Documentos

Para atualizações das informações do documento adicionado ao repositório, basta marcar o arquivo, selecionar “Editar” e preencher os campos informados anteriormente.

 

A opção permite que o editor substitua um documento já adicionado ao repositório por um novo arquivo ou outra versão do mesmo documento. Caso queira desfazer da escolha do arquivo, basta acionar . Transformação de Arquivos

O Lumis disponibiliza a opção de se realizar transformações de tamanho/formato sobre os arquivos adicionados ao repositório. Para informações sobre Transformações de Arquivos, consultar a seção Transformação de Arquivos.
