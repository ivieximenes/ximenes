Tela Criar Módulo Editável

 

Esta tela permite a criação de módulos editáveis.

 

 

 Cria um módulo editável.

 

 Campos:

 - Nome: Nome do módulo novo.

 - Descrição: Descrição do módulo novo.

 - Id: Identificador do módulo.

 - Diretório do código-fonte: Diretório onde os arquivos do novo módulo serão criados, relativo a "lumisdata/shared/source".
