Através deste Gerenciador é possível visualizar os web resources do portal gerados, principalmente as urls amigáveis e caches de página gerados, ou seja, tudo aquilo que pode ser acessado via URL do portal.

Os registros de web resources acima correspondem a urls de páginas configuradas para cache e urls amigáveis, por exemplo.

 

A partir do acesso à página (dentro ou fora do F2) que possui url amigável, cache ou os dois configurados, é exibido UM registro no gerenciador de web resource. Com execeção de págians que recebem parâmetros. São registrados um para cada parâmetro. Vide a url da página de detalhes na figura acima.

 

Também são exibidos os registros dos ancestrais que possuem a propriedade de urls amigáveis configurada e todos os registros de páginas dependentes entre si quando o cache de página está habilitado.

 

Através do gerenciador é possível excluir um ou vários registros de cada vez ou todos. A exclusão é necessária no caso de alteração na configuração da url amigável, em que o registro no gerenciador de web resource não é alterado ou removido automaticamente. Além disso, o portal não adicionará as novas entradas para as urls alteradas e continuará servindo as antigas caso elas não sejam removidas.

 

É importante ressaltar que o canal ou página da estrutura não são removidos ao excluir o registro da url no gerenciador de web resource.

 

Um registro somente é removido automaticamente quando o canal ou página são removidos da solução.

 

Observação

 

Recomenda-se não desabilitar e nem excluir os observadores padrão, pois pode influenciar no correto funcionamento do produto. Ao desabilitar a configuração de url amigável ou cache de página, o cadastro no web resource não  é removido.
