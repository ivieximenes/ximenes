Administração de Auditoria

 Top  Previous  Next

Na seção “Configurações / Informações / Auditoria / Administração de Auditoria” é possível realizar tarefas administrativas relativas à Auditoria.

A interface “Tipos” permite listar os Tipos de Auditoria filtrando por Categoria, Nome e Habilitado. Este último é o campo que define se os tipos de Auditoria devem ser salvos no banco de dados.

Através desta interface é possível habilitar e desabilitar os tipos de Auditoria. Para habilitar um determinado tipo, basta selecioná-lo e clicar em “Habilitar”. O mesmo vale para a ação de desabilitar. Mesmo os tipos desabilitados terão as operações registradas no arquivo de log.

Além de habilitar e desabilitar os tipos de Auditoria, também é possível configurar no Serviço de Auditoria o tempo que entradas serão armazenadas. Na tela de administração, ao selecionar a opção “Configuração”, a tela de configuração será exibida:

Após o tempo especificado (em dias) as entradas de Auditoria serão removidas do banco de dados.
