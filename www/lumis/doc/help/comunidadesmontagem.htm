O serviço pode ser montado de diversas maneiras, para o usuário com perfil de administrador do serviço e para os usuários com perfis de criar e visualizar as comunidades, dependendo de como foi arquitetado pelo administrador do portal.

Para utilizar o serviço de Comunidades, é necessário instanciá-lo em um canal, adicionar a este canal um template de canais que será utilizado no cadastro de novas comunidades, e arrastar suas interfaces para as áreas das páginas em modo de edição.

Suas interfaces correspondem a um ambiente que permite a administração do serviço, o cadastro de novas comunidades, a visualização das comunidades cadastradas e de seus detalhes, o acesso às páginas das comunidades, a visualização dos membros de uma comunidade e a administração da comunidade e de seus membros.

Observação

 

O canal que contem a instancia do serviço e o template de comunidades não deve ser visível por usuários do portal pois a permissão é herdada pelos canais criados para as comunidades.

No entanto, qualquer usuário que vá ter acesso a comunidades deve ter permissão de visualizar a instancia do serviço de comunidades e ter acesso as páginas com as interfaces "Todas as Comunidades", "Minhas Comunidades" e "Detalhes" de comunidade.

Abaixo, segue a descrição de cada uma das suas interfaces:

•Todas as Comunidades: Esta interface disponibiliza uma lista de todas as comunidades cadastradas com privacidade pública ou restrita, exibindo a imagem, o título e a descrição de cada comunidade;

Interface "Todas as Comunidades"

•Minhas Comunidades: Esta interface disponibiliza uma lista de todas as comunidades das quais o usuário é membro. Para cada comunidade dessa lista, é exibindo a sua imagem, o seu título e a sua descrição. Além disso, para os usuários que possuem permissão de criar comunidades, também é disponibilizado um botão que possibilita esta ação;

Interface "Minhas Comunidades"

•Detalhes: Esta interface pode ser acessada por usuários que não participam de uma comunidade, ao clicar no ícone ou no título dessa comunidade. Esta interface exibe o ícone, o título, a descrição e o nível de privacidade da comunidade e disponibiliza um botão para participar da comunidade, ou requisitar a participação, caso seja uma comunidade restrita;

Interface "Detalhes"

•Cabeçalho: Esta interface exibe o título, o ícone a descrição e o botão de sair da comunidade que está sendo acessada, para os membros da comunidade que possuem papel de administrador da comunidade, também são disponibilizados botões para editar e excluir a comunidade;

Interface "Cabeçalho"

•Membros: Esta interface disponibiliza uma lista de todos os membros de uma comunidade. Para cada usuário dessa lista, é exibido a sua imagem, o seu nome e o seu papel dentro da comunidade. Para os administradores da comunidade também são disponibilizadas as opções de adicionar membros à comunidade e remover ou alterar o papel de membros da comunidade.

Interface "Membros"

•Administração: Esta interface disponibiliza para os administradores do serviço o cadastro de novas comunidades e a edição ou a exclusão das comunidades cadastradas.

Interface "Administração"

 

O template de canal criado para ser usado no cadastro de novas comunidades, deve ter a opção de grupos locais habilitada. Para isso, deve-se acessar as opções de segurança nas propriedades avançadas do template e selecionar o checkbox de Habilitar Grupos Locais.

Propriedades Avançadas do template

Após ter habilitado a utilização de grupos locais, deve-se adicionar a este template os grupos "Members" e "Administrators". Essa operação pode ser feita na gerência de grupos locais do template, clicando em adicionar.

Para a criação do grupo "Members", deve-se adicionar um grupo preenchendo o campo apelido com "members" e o campo nome com "Members".

Para a criação do grupo "Administrators", deve-se adicionar um grupo preenchendo o campo apelido com "administrators" e o campo nome com "Administrators".

Gerência de Grupos Locais

Para disponibilizar o serviço de Comunidades para o usuário final, deve-se instanciar as interfaces Cabeçalhos e Membros em alguma página dentro do template de comunidades. Já, as interfaces Todas as Comunidades, Minhas Comunidades e Detalhes, devem ser instanciadas fora do template de comunidades, em quaisquer diferentes páginas do portal.

 

Observação

 

O Lumis Portal oferece como sugestão de montagem deste serviço um prebuilt channel com uma estrutura de canal, páginas, e disposição das interfaces nas páginas. Após adicionar o prebuilt channel, os controles de acessos devem ser devidamente ajustados.

Para informações sobre como instanciar canais, páginas e atribuição de permissões no controle de acesso, consultar a seção sobre Portal Studio.

Através do right-click da interface, encontram-se disponíveis parametrizações para a instância de serviço e customizações para a interface. Consultar a seção Customização de instâncias de Interfaces.
