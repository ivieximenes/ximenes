Consumer Configured Portlet

 Top  Previous  Next

Consumer Configured Portlets são portlets wsrp onde existe uma configuração por parte do consumidor para sua customização.  Esta configuração só é possível em portais que dão suporte à Registro.  Para maiores informações sobre a API, acesse a especificação.

 

Para realizar esta configuração é necessário que exista uma instancia de Portlet Remoto configurado com registro para um produtor.

 

Clonando um Portlet:

1) Click com o botão da direita em cima do serviço de Portlet Remoto previamente configurado e selecione Configurar Clones.
 

 

2) A tela com a lista de portlets disponíveis abrirá.

3) Selecione o portlet que deseja clonar e click no botão Clonar Portlet.
 


 

4) A tela com informações sobre o clone abrirá.  Preencha o campo de Nome amigável.  Para cada propriedade a ser configurada no clone, caso haja alguma, aperte o botão Adicionar para criar um novo par Chave-Valor.  Preencha os valores conforme desejado.  Ao final da configuração, pressione Ok.
 


 

Observação:

O portlet remoto deverá ser capaz de entender as propriedades configuradas.  Caso contrário, estas informações serão armazenadas no servidor remoto e ignoradas pelo portlet.

 

5) O novo clone deve estar disponível na lista de portlets disponíveis.

 

Alterando Propriedades de um Clone:

1) Click com o botão da direita em cima do serviço de Portlet Remoto previamente configurado e selecione Configurar Clones.

2) A tela com a lista de portlets disponíveis abrirá.

3) Para abrir a tela de detalhes do clones clique no link do clone.  Portlets que não são clones, não terão link.

4) A tela de Detalhes do Clone será exibida.  Esta tela possui as informações de nome amigável, em quais páginas o clone está sendo utilizado e quais as propriedades configuradas.
 


 

5) Faça os ajustes nessessários nas propriedades do clone e aperte Ok.  A alteração afetará todas as páginas que utilizarem o clone em questão.

 

Removendo um Clone:

1) Click com o botão da direita em cima do serviço de Portlet Remoto previamente configurado e selecione Configurar Clones.

2) A tela com a lista de portlets disponíveis abrirá.

3) Selecione o clone que deseja excluir e aperte no botão Excluir Clone.
