É possível adicionar arquivos de CSS customizados que podem sobrescrever o portal.css (arquivo de css default do portal). Podem ser customizados estilos CSS referentes a fontes, cor de fundo da página, etc. que serão repassados para as suas páginas, caso a herança nelas esteja habilitada.

 

 

•CSS: Permite adicionar arquivos CSS que contêm formatações para o canal,  para seus canais filhos e páginas. A figura anterior informa que o canal “Agenda de Eventos” está utilizando as formatações de CSS do arquivo “intranet.css ” que sobrepõem àquelas definidas no arquivo “portal.css”. Podem ser adicionados tantos arquivos quanto necessário. Sendo que os últimos adicionados ficam localizados no final da lista e sobrepõem as formatações dos primeiros. Verifique que a ordem dos arquivos pode ser modificada através das ações Mover para cima e Mover para baixo. Caso necessário, pode-se excluir arquivos CSS adicionados;
 
•Herdar utilização de CSS: É permitido herdar as configurações CSS de uma lista de arquivos. Esta lista corresponde aos arquivos CSS adicionados a canais superiores. O último arquivo da lista tem prioridade sobre o penúltimo e assim por diante. Caso a herança seja desabilitada, o canal adotará somente os estilos CSS adicionados para ele. Considerando a figura acima, caso a herança seja desmarcada, o canal “Interna” adotará somente as configurações definidas no arquivo “intranet.css”.

 

Observações

 

1) Os novos arquivos CSS adicionados devem estar previamente registrados no portal. Para maiores informações, consultar a seção Gerenciador de CSS.

2) A exclusão de arquivos CSS do Estilo não remove o registro do mesmo no portal.
