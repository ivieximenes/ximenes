Script de instalação

 Top  Previous  Next

Esse script será executado quando o componente for instalado. Esse arquivo é um arquivo no formato XML que deve seguir o esquema lumis-component-install.xsd, cujo link está presente em API e XSD. Além desse script, o portal pode executar passos adicionais, como:

•Registro de serviços
•Registro de temas
•Registro de contextos de negócio
•Registro de observadores
•Cópia de arquivos
•Importação de estrutura

 

Esses passos adicionais são passos automaticamente tratados pelo portal e, portanto, não necessitam estar explicitamente descritos no script. É importante notar, no entanto, que esses passos serão tomados para os recursos pertencentes ao componente em questão. Mesmo que o componente não possua esse script, os passos adicionais serão executados pelo portal.

 

Os tipos de passos disponíveis para o script de instalação

O script de instalação disponibiliza os seguintes tipos de passos:

•Executar comando SQL: Executa um comando SQL no banco de dados.
•Executar arquivo de definição de banco e dados: Toma como entrada um arquivo de definição de banco de dados e executa os comandos gerados a partir desse arquivo.
•Executar uma classe Java: Executa uma classe Java no servidor.
•Exibir uma mensagem: Exibe uma mensagem informacional.
•Limpeza de itens globais não referenciados: Remove itens globais (estilos globais de instâncias de interface, registros de CSS e parâmetros de transformação de arquivos) não referenciados.

 

Observação

 

Caso o componente não possua script de instalação o seu script de atualização (caso exista) será executado por completo ao realizar-se a instalação do componente.
