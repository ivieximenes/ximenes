Arquivo de Definição

 Top  Previous  Next

Um arquivo de definição de web service, ou WSDL (Web Service Definition Language), contém todas as informações necessárias para que uma aplicação cliente possa conectar-se a um Web Service e utilizar suas funcionalidades.

No Lumis Portal, os WSDLs podem ser acessados a partir do endereço de cada Web Service, seguido de "?wsdl". Por exemplo, para o Web Service de autenticação, o WSDL pode ser obtido no endereço:
 

http://<ENDEREÇO_DO_PORTAL>/webservice/AuthenticationService?wsdl

 

Lista de webservices:
 

AuthenticationService
 

URL do WSDL:
 
http://<ENDEREÇO_DO_PORTAL>/webservice/AuthenticationService?wsdl
 

UserService
 

URL do WSDL: http://<ENDEREÇO DO PORTAL>/webservice/UserService?wsdl
