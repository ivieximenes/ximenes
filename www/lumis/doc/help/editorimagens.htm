Editor de Imagens

 Top  Previous  Next

Nos campos do tipo arquivo, quando o arquivo escolhido se tratar de uma imagem e esta imagem já estiver salva, será exibido um link para uma interface de edição de imagem.

 

Ao acessar essa interface, o usuário pode aplicar zoom, rotacionar e selecionar uma área de corte na imagem escolhida, gerando um novo arquivo de imagem com as edições realizadas, que passará a preencher o campo de arquivo onde foi escolhida a imagem original.

Ao retornar da interface de edição de imagem, caso o usuário tenha aplicado as edições realizadas, o campo de arquivo passará a exibir um link para que o usuário possa descartar o arquivo contendo a imagem editada e utilizar um outro arquivo.

 

Na interface de edição de imagem são disponibilizadas para o usuário as seguintes funcionalidades:

 

Campos para fixar as dimensões da área de seleção para cortar a imagem. Só são permitidos números inteiros maiores do que zero.

Botões para aplicar zoom e rotacionar a imagem.

Nesse modo, ao clicar com o mouse em uma área da imagem que não esteja selecionada e arrastar, a imagem será movida.

Nesse modo, ao clicar com o mouse em uma área da imagem que não esteja selecionada e arrastar, uma nova área será selecionada.

Desfaz todas as alterações de zoom, rotação e movimentação da imagem e cria uma nova área de seleção.
