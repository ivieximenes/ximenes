O serviço pode ser montado de diversas maneiras, dependendo de como foi arquitetado pelo administrador do portal.

Para utilizar o serviço de Impressão, basta instanciá-lo em um canal e arrastar sua interface para as áreas das páginas em modo de edição.

 

•Imprimir página: Esta interface permite imprimir a página atual em uma nova janela escondendo as interfaces de serviços que foram configuradas para não serem impressas. Além disso, executa a função de impressão do Browser.

Observação

 

Para informações sobre como instanciar canais e páginas, consultar a seção sobre Portal Studio .

Parametrização da Instância de Serviço

Algumas customizações do serviço de Impressão são acessíveis através do right-click na interface Imprimir página que devem ser configuradas pelo administrador do portal. Dentre elas está a customização Configurações de Impressão que permite definir se o conteúdo será impresso a partir de uma nova janela do browser ou não, utilizando sempre as configurações de impressão do browser.

Parametrização da Instância de Interface

Em páginas que estão configuradas para serem cacheadas, e nas quais a interface Imprimir Página está instanciada, o estilo "Imprimir como página HTML"  deve ser seleciondo pelo administrador do portal para esta interface. Este procedimento garante que a impressão da página cacheada ocorrerá devidamente.

No gerenciador de cache serão exibidos duas páginas cacheadas, uma com impressão e outra sem.

Observação

 

Através do right-click da interface, encontram-se disponíveis parametrizações para a instância de serviço e customizações para a interface. Consultar a seção Customização de instâncias de Interfaces.
