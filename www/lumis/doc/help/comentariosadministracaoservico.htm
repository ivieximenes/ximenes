O serviço Comentários possui uma interface administrativa que permite que os comentários sejam aprovados antes de serem publicados, caso seja um serviço moderado.

A figura abaixo exibe a interface administrativa com "todos" os comentários já postados por diversos usuários. Através dessa interface é possível excluir os comentários. Consequentemente, eles serão excluídos das listas de comentários de determinado conteúdo ao qual pertence.

Através desta interface também é possível configurar se os comentários serão moderados ou não. Determinar se os comentários serão moderados, significa que, para serem publicados, precisam da interferência do administrador do serviço.

O valor default é "não". Desta forma, todos os comentários postados são considerados como aprovados (status "Aprovado"), e portanto, automaticamente são publicados nas listas de comentários.

Assumindo que o serviço está configurado como moderado (valor "Sim"), todos os comentários postados são registrados na interface administrativa com o status "Pendente". Ou seja, que ainda não passaram pela aprovação do administrador do serviço.

Para aprovar um comentário, o administrador do serviço deve selecioná-lo na interface administrativa e clicar em "Editar". A figura abaixo é um exemplo da edição de um registro:

Ao editar, é permitido ao administrador do serviço, alterar a nota atribuída pelo usuário e alterar o comentário postado para que seja aprovado (atribuir o valor "Aprovado" ao campo Status ).

O campo "Status" possui os seguintes valores:

•Pendente: status que é atribuído automaticamente ao comentário quando ele é postado, quando o serviço está configurado como moderado. Com este status , o comentário não é publicado nas listas. Pode ser alterado para outro status , conforme conveniência;
•Aprovado: status atribuído ao comentário para que seja publicado. Quando o serviço não está configurado como moderado, todos os comentários postados possuem este status.  Pode ser alterado para outro status, conforme conveniência;
•Reprovado: status atribuído ao comentário para que não seja publicado. Uma vez que o comentário esteja aprovado, ao alterar o status para "Reprovado", o comentário não fará parte das listas de comentários;
•Arquivado: este status é conveniente quando o comentário for passar por uma revisão pelo administrador do serviço. Desta forma, o comentário não é visualizado nas listas de comentários.

É disponibilizado para o administrador do serviço uma lista dos resultados das médias das notas publicadas para cada conteúdo. Essas médias, nada mais são que a soma das notas publicadas, dividido pelo número de votos.
