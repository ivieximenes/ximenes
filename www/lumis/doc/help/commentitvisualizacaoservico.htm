Visualização do Serviço

 Top  Previous  Next

Para o usuário que tenha permissão de visualizar o serviço CommentIt, para cada conteúdo da página que tenha sido selecionado pela query no xsl da interface "CommentIt Scripts", é exibido o total de comentários feitos para o conteúdo, uma lista com os últimos 3 comentários para este conteúdo e um link para exibir mais 5 comentários a cada vez que é clicado.

Abaixo da lista de comentários é disponibilizada uma caixa de texto para o usuário escrever seu comentário e postá-lo ao pressionar a tecla Enter.

Para excluir o próprio comentário, basta que o usuário passe o ponteiro do mouse sobre o próprio comentário para ser disponibilizado um link que permite excluí-lo.
