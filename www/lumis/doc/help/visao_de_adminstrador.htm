Perfil de Administrador

 Top  Previous  Next

Este perfil pode ser acessado através da tecla de função F2, ou pelas opções de perfil do portal, após os usuários com perfis de administradores efetuarem o login no portal. Somente usuários com este tipo de perfil podem acessar o perfil de Administrador:

Este perfil é detentor de dois modos, o Portal Studio e Configurações. Neles são possíveis gerenciar todo o Portal Studio ou apenas um fragmento deste, dependendo das permissões concedidas ao usuário. Para maiores informações sobre permissionamento e gerenciamento do portal studio visite, Controle de Acesso do Portal e Portal Studio.

 

Navegação entre Visões:

A partir deste perfil, é possível navegar para o perfil de Publicador (se permitido) através do dropdown dos modos disponibilizados e para o perfil deste Usuário, que além de ser acessível pelo dropdown de modos, é acessível também pressionando a tecla F2.

 

Observações

 
•Caso o usuário não tenho o perfil de administrador, a Visão exibirá desabilitada
•Não é gerado cache nesta visão.
