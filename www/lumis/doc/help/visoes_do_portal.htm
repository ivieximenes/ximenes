Perfis do Portal

Perfis do Portal são perspectivas de acesso ao Lumis Portal. Através delas, o Lumis pode representado por diferentes formas, cada uma delas, com diferentes modos únicos. Usuários terão acesso apenas aos modos para os quais possuem permissão. Os Perfis disponibilizadas são de Administrador, Publicador e Usuário.

Navegação entre Perfis:

 
Para trocar para outro perfil desejado, é necessário acessar o dropdown à esquerda da barra e selecionar o modo desejado. Caso algum destes não esteja permitido ao usuário, o mesmo se encontrará desabilitado.
