Versão 5.5.0

 Top  Previous  Next

Novidades

 

•Serviço Produtor WSRP, que disponibiliza instâncias de interfaces como portlets remotos;

 

•Serviço Consumidor WSRP, consume portlets remotos;

 

•O Lumis Portal dá suporte para indexação de arquivos nos formatos: texto (.txt), PDF, HTML, arquivos do Microsoft Office e arquivos do OpenOffice;

 

 

Melhorias

 

•Adição no displayName em instância de interface o campo nome. Para maiores informações, consulte o tópico Customização de Instância de Interface e/ou Serviço;

 

•Melhoria de performance criando um pool de DocumentBuilder para ser usado pelo XmlUtil. Para maiores informações, consulte o tópico Configurações do Portal ;

 

 

Alterações

 

•Alteração no algoritmo padrão de criptografia para SHA-512. Em ambientes com versões anteriores do Lumis Portal, o algoritmo padrão era o SHA-1, portanto, para realizar o upgrade desses ambientes poderá ser necessário configurar o algoritmo de criptografia a ser utilizado. Para mais detalhes consulte o tópico Configurações do Portal. Por exemplo, as senhas dos usuários armazenadas internamente na base do Lumis Portal usam essa configuração.

 

•Parâmetros disponíveis no PageWebresource passaram a ser decodificados.

 

•Alteração no controle de lum_form. O action do elemento form foi alterado para sempre ser gerado como um actionURL.

 

•Alteração no DOUI ao processar um actionURL. Caso a requisição não possua o parâmetro doui_processActionId, nenhum processAction será executado e os parâmetros serão repassados como parâmetros de renderização.

 

•Ambientes suportados:

 - A partir dessa versão, não há mais homologação e suporte para a configuração do Lumis no Sun J2SE 5.0;

 - A partir dessa versão, não há mais homologação e suporte para a configuração do Lumis no servidor JBoss Application Server 4.0.4.;
