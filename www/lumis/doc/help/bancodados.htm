Neste tópico são descritos os passos necessários para a criação do banco de dados do Lumis Portal no servidor de banco de dados.

 

Primeiramente, defina as variáveis de ambiente “LUMIS_HOME” e “JAVA_HOME” ou no arquivo setup/setclasspath.cmd, caso seja necessário. O exemplo abaixo utiliza os caminhos absolutos de instalação do ambiente.

 

setclasspath.cmd:

 

SET LUMIS_HOME=C:/lumisportal_numero_da_release

if "%JAVA_HOME%" ==  "" (SET JAVA_HOME=C:/Arquivos de programas/Java/jdk1.8.0_25)

 

setclasspath.sh (se o ambiente for o linux):

 

[ -z "$LUMIS_HOME" ] && LUMIS_HOME=/lumispotal_numero_da_release

[ -z "$JAVA_HOME" ] && JAVA_HOME= /java/jdk1.8.0_25

 

De acordo com a versão utilizada do servidor de banco dados, acesse a seção correspondente, para verificar as instruções de como gerar o script SQL e criar o banco.

 

Observação

 

Na configuração do setclasspath.cmd ou setclasspath.sh, verificar o diretório em que o Lumis está localizado e a instalação do Java.
