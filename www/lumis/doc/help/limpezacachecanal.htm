Limpeza de Cache

 Top  Previous  Next

Esta opção, disponível nas propriedades do canal através do botão direito no navegador, é um acesso rápido para limpar o cache de memória:

•dos objetos utilizados internamente pelo canal;
•dos XSLs utilizados pelo canal e pelas suas páginas;
•das interfaces (aquelas que foram configuradas para serem cacheadas) instanciadas nas páginas do canal;
•das strings ;
•da ACL (Access Control List );
•entre outros.

Além da limpeza de cache de memória de tais objetos, ocorre a limpeza de cache HTML das páginas (aquelas configuradas para serem cacheadas) do canal.

Os únicos canais que não têm o cache limpo são aqueles aos quais o usuário não possui acesso, e seus respectivos subcanais.

Observações

 

Existe diferença entre limpar cache no canal portal e cache de canal interno. A limpeza de cache no canal portal, remove os caches de memória, todos os caches de páginas e interfaces. Já a limpeza de cache de um canal interno são limpos todos os caches de páginas e interfaces na hierarquia dentro daquele canal, havendo portanto um desempenho menor. Os caches de memória, neste último caso não são afetados.

 

Nos dois casos,  os servidores, em um ambiente em cluster, são sincronizados após a limpeza.
