Redes Sociais Privadas

 Top  Previous  Next

Nesta seção é possível gerenciar redes sociais privadas do portal. Essas redes definem informações básicas (como canal base, nome e página pessoal) e informações sobre o provedor de informações dos perfis sociais dos usuários.

Rede Social Privada

Uma rede social privada é definida por:

•Nome: Nome da rede social provada;
•Página Pessoal: Página de perfil social. Essa página servirá para exibição do perfil social de um dado usuário. Essa página será usada para definição de um link para o perfil pessoal de um usuário. Essa página deve, por padrão, receber o parâmetro de nome lumUserId, contendo o identificador do usuário que terá seu perfil exibido.
•Canal Base: É o canal base dessa rede social privada. Todos os canais descendentes desse na hierarquia de canais utilizará essa rede social como sua rede social, a menos que uma nova rede seja criada em um desses canais descendentes. É importante salientar que duas redes sociais não podem utilizar o mesmo canal base.
•Provedor de informações para perfil social: é a fonte de dados que proverá os dados dos perfis sociais dos usuários.

Provedor de informações para perfil social

Esse provedor é responsável por prover as informações dos perfis sociais dos usuários. Dois tipos de provedores são suportados pelo Lumis Portal:

•Classe: Uma classe customizada que implementa a interface ISocialProfile.
•Instância de serviço: Para utilizar uma instância de serviço, o serviço dessa instância deve ser marcado como provedor de perfil social (adicionando o nó <isSocialProfile>true</isSocialProfile> ao nó de definição do serviço no service definition). Além da instância de serviço, esse tipo de provedor necessita de um source e o identificador do campo (nesse source) no qual o identificador do usuário é armazenado.

Gerenciamento de Redes Sociais Privadas

Através do gerenciador de redes sociais é possível:

 

Adicionar

Permite adicionar uma nova rede social.

 

 

 

Onde:

•Nome: Nome da rede social.
•Páginal Pessoal: Página do portal que será utilizada para exibir as informações do perfil social.
•Canal Base: Canal do portal que irá servir de base para rede social.

 

Mais abaixo é encontrada uma área para configurar o provedor de informações do perfil social onde serão dispostas as seguintes opções:

 

•Serviço: Nessa opção, a rede social será configurada a partir de uma serviço.

 

Serviço: Serviços de perfil social. É o serviço que servirá de base para os dados de perfis sociais do usuário.

Instância de Serviço: Instâncias do serviço selecionado. É a instância da onde devem ser lidos os dados de perfils sociais dos usuários.

Source: Sources da instância de serviço selecionado. É o source de onde devem ser lidos os dados de perfils sociais dos usuários.

Campo: Campos do source selecionado. É o campo que armazena o identificador do usuário. Esse campo é utilizado para filtrar o perfil do usuário.

 

•Classe: Nessa opção, a rede social será configurada a partir de uma classe java.

 

 Classe: Nome completo da classe que será utilizada para customização do provedor de dados.

 

 

 

Editar

Permite editar as informações previamente adicionadas de uma rede social.

 

 

Remover

Permite a remoção da(s) rede(s) social(is) selecionada(s).
