Administração do Conteúdo

O serviço de Enquete possui um ambiente administrativo disponibilizado pelo administrador do portal, para o cadastro do conteúdo pelo editor responsável. Uma vez o conteúdo cadastrado, é possível alterá-lo, visualizá-lo ou excluí-lo.

 

•Pergunta: Pesquisa que estará disponível ao usuário para a votação;
•Usuário responde apenas uma vez: o preenchimento deste campo possui comportamento diferente quando o serviço estiver disponibilizado em uma área restrita (para usuários cadastrados) ou em uma área pública.

Caso o serviço de Enquete esteja disponibilizado em uma área restrita, o preenchimento deste campo determinará que o usuário cadastrado só poderá votar uma única vez por enquete. Este controle é feito juntamente com o login do usuário. Se este campo não for habilitado, não haverá restrição para quantidade de votação por login.
 

•Tipo de escolha: Permite determinar se a enquete terá mais de uma resposta (escolha múltipla) ou não (escolha única).
•Número de opções: Permite determinar o número de respostas que serão cadastradas para a enquete;
•Opção X: permite cadastrar uma das repostas da enquete;

O serviço de Enquete permite que o editor de conteúdo estabeleça regras para publicação como período que determinado conteúdo estará disponível para o usuário final da solução. Permite também publicar o conteúdo para determinados grupos e para outras instâncias do serviço de Enquete (vantagem de cadastrar o conteúdo somente uma vez e fazê-lo com que seja conteúdo de outras instâncias do mesmo tipo de serviço).

Por ser um serviço de gestão de conteúdo, apresenta a funcionalidade de workflow para aprovação dos conteúdos cadastrados.

 

As enquetes cadastradas formam um conjunto de registros para posteriores manutenções em seus conteúdos. A disponibilização dos campos que aparece na figura abaixo depende de como o administrador do portal configurou o ambiente administrativo de cadastro de conteúdo de enquetes.

A enquete que será disponibilizada para votação, será a mais recentemente cadastrada ou alterada, que está aprovada no processo de workflow e dentro do período de publicação.

O editor de conteúdo pode acompanhar os resultados da votação, através do ambiente administrativo, selecionando a opção “Resultados”, para cada enquete cadastrada.
