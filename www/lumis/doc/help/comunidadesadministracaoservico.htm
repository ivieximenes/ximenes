O serviço de Comunidades possui um ambiente administrativo disponibilizado pelo administrador do portal, que permite o cadastro de novas comunidades e a edição ou a exclusão das comunidades cadastradas.

Interface de administração de comunidades

Essa interface administrativa, só pode ser acessada por administradores do serviço que tenham a permissão de Gerenciar Instância de Serviço.

Ao adicionar uma comunidade, o administrador se torna o único membro e administrador dessa comunidade. Para criá-la, o administrador deve, obrigatoriamente, informar o nome e o nível de privacidade da comunidade que deseja criar e pode, opcionalmente, adicionar uma descrição e uma imagem para essa comunidade.

São disponibilizados três níveis de privacidade para as comunidades:

•Público: As comunidades com nível público de privacidade são exibidas na lista de todas as comunidades e qualquer usuário pode participar delas;
•Restrito: As comunidades com nível restrito de privacidade são exibidas na lista de todas as comunidades, mas para participar delas os usuários devem solicitar participação para os administradores da comunidade;
•Privado: As comunidades com nível privado de privacidade não são exibidas na lista de todas as comunidades e os usuários só podem participar delas se forem adicionados diretamente por algum administrador da comunidade.

 

Para ver mais detalhes do papel de um administrador de uma comunidade, veja o tópico de visualização do serviço.

Com o cadastro de uma comunidade, é criado um canal com o nome cadastrado para a comunidade, a partir do template de canais para as comunidades criado pelo administrador do portal, dentro do canal onde o serviço de comunidades está instanciado.

Ao editar o cadastro de uma comunidade, o administrador pode alterar o nome, a descrição, a imagem e a privacidade da comunidade. Contudo, ao editá-la, não é possível administrar os seus membros.

Observação

 

Não é possível remover o canal utilizado por uma comunidade pelo Portal Studio. Deve-se utilizar a interface de administração e excluir a comunidade.

Para um administrador do serviço de comunidades administrar os membros de uma comunidade, ele deve acessar a lista de membros da comunidade.
