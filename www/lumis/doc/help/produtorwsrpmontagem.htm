O serviço pode ser montado de diversas maneiras, para os administradores do serviço, dependendo de como foi arquitetado pelo administrador do portal.

 

Para utilizar o serviço de Produtor WSRP, basta instanciá-lo em um canal e arrastar sua interface para a área das página em modo de edição. Sua interface corresponde a um ambiente administrativo.

 

•Administração do Produtor: Esta interface permite configurar portlet remoto e cadastrar instâncias de interface.
 
•Gerenciar Registros: Esta interface permite adicionar registros outbound, extender lifetime e excluir registros.

 

•Detalhes do Registro: Esta interface permite visualizar os detalhes de um registro.  Um link é gerado da interface de Gerenciar Registros para esta interface uma vez que ela esteja instanciada.
 

O ambiente administrativo do serviço Produtor WSRP requer uma página para cada interface em um canal restrito do portal, onde apenas os editores de portlet terão acesso, após o login. A fim de proporcionar uma maior organização no portal, é aconselhável que esta página pertença a um canal específico, o qual está sendo chamado aqui de “Produtor WSRP”.
 


 

 

Atribuição de Permissões

 

Para cadastrar, configurar e publicar portlet, o administrador do portal deve atribuir a permissão "Visualizar instância de serviço". Esta permissão é concedida no controle de acesso da instância do serviço.

 

Observação:

A permissão na instância de serviço do produtor WSRP é necessária apenas para quem irá acessar sua administração.

Esta permissão não é necessária para visualizar as interfaces que o produtor oferece por WSRP. Neste caso o necessário é permissão na instância de serviço da interface sendo exposta por WSRP.

 

Na figura a seguir, há um exemplo do controle de acesso de uma instância de serviço, cujo usuário “Andresa” está com permissão “Visualizar instância de serviço”, atribuída pelo administrador do portal. Com a permissão “Visualizar instância de serviço” o usuário tem acesso a interface administrativa e conseguirá cadastrar e configurar portlet.
 

 

Observação:

Para informações sobre como instanciar canais, páginas, e atribuição de permissões no controle de acesso, consultar a seção sobre Portal Studio.

Através do right-click da interface, encontram-se disponíveis parametrizações para a instância de serviço e customizações para a interface. Consultar a seção Customização de instâncias de Interfaces.
