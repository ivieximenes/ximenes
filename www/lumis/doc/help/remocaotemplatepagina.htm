Remoção de Template de Página

 Top  Previous  Next

A remoção de um template de página excluir todos os dados associados a ele, como definição das páginas e dados das instâncias dos serviços. Porém os dados da página que o contém, não são removidos.

Existem duas formas de remover um template de página:

•Com o botão direito sobre o template de página que deseja remover, selecione a opção Excluir;

 

Ou

•Selecione o canal “Portal” ou um outro canal existente;
•Na área administrativa do canal, selecione a opção Template de Página, em seguida, selecione o(s) template(s) desejado(s), por fim, selecione Excluir.
