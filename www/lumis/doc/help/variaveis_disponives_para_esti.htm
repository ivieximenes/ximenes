lum_serviceInterfaceRequest

É um objeto do tipo IServiceInterfaceRenderRequest.

Contem informações da requisição(request) sendo usado na interface sendo gerada.

lum_serviceInterfaceResponse

É um objeto do tipo IServiceInterfaceRenderResponse.

Contem informações de resposta(response) usados pela interface sendo gerada.

lum_out

É um objeto do tipo PrintWriter.

Criado a partir do response, permite acessar diretamente a interface de criação do arquivo HTML sendo gerado.

lum_el

É um objeto do tipo ELEvaluator.

Criado a partir do request, disponibiliza aos scripts de renderização dos estilos os recursos em EL.

lum_document

É um objeto do tipo Document.

Criado a partir do XML gerado para a instancia atual da interface, é o contexto principal para renderização da mesma.

lum_xpath

É um objeto do tipo IXPath.

Disponibiliza vários métodos para auxiliar a leitura do contexto da interface. Para maiores informações veja o tópico Variável lum_xpath.
