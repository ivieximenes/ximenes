Histórico de pacotes

O portal mantém um histórico dos pacotes instalados no portal. A interface é exibida como na figura abaixo:

Histórico de pacotes

 

Nessa interface pode-se também ativar o botão "Instalar" para chamar a interface de deployment de pacotes, que pode ser vista abaixo.

 

Deployment de pacotes

Nessa interface pode-se fazer o upload de um arquivo de pacote ou selecionar um arquivo de pacote previamente enviado para o servidor, como mostrado na figura abaixo.

Seleção de arquivo de pacote

 

Após selecionar o arquivo (seja ele enviado no momento, seja ele escolhido um arquivo pré-enviado) o usuário será redirecionado para a interface de confirmação de deployment, onde poderá revisar todas as ações que serão tomadas antes de prosseguir com o deployment.

Ao realizar a cópia do arquivo de pacote para que o mesmo possa ser utilizado em um deployment, o administrador deve copiá-lo para a pasta lumisdata/deploy/packages, em cada servidor do cluster.
