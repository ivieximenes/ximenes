O serviço de Conteúdo HTML possui um ambiente administrativo, disponibilizado pelo administrador do portal, para o cadastro do conteúdo pelo editor responsável. O conteúdo cadastrado pode ser alterado, visualizado ou excluído.

Interface de administração de conteúdos HTML

 

Para adicionar ou editar um conteúdo, utiliza-se a interface de Adicionar e Editar representada abaixo

Interface de adicionar e editar conteúdo HTML

 

•Título: Título (obrigatório) que identifica o conteúdo na área administrativa;
•Conteúdo: Texto (obrigatório) interno. O cadastro deste campo é realizado em um editor de texto que permite tipos de formatação semelhantes aos editores de texto conhecidos. Pode-se ainda editá-lo em HTML através da opção “Código-Fonte” do editor de texto;

O serviço de conteúdo HTML utiliza os sistemas de Workflow, Publicação de Conteúdos, Versionamento e Publicação Multilingue do portal.

 

Observação

 

Para informações sobre período de publicação, workflow, versionamento, publicação para outros idiomas e visualização do conteúdo, consultar a seção “Funcionalidades Gerais dos Serviços de Gestão de Conteúdo”.
