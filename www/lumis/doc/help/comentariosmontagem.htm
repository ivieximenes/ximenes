Montagem

O serviço pode ser montado de diversas maneiras, dependendo de como foi arquitetado pelo administrador do portal.

Para utilizar o serviço Comentários, basta instanciá-lo em um canal e arrastar suas interfaces para as áreas das páginas em modo de edição:

•Lista Rápida: Esta interface lista alguns comentários. A quantidade de comentários exibidos nesta lista, depende do "Número de Itens" informados pelo administrador do portal, no right-click desta interface. A partir desta interface, o usuário pode ainda cadastrar comentários. A interface Lista Rápida deve ser instanciada juntamente com a interface de Detalhes da instância de serviço, cujo conteúdo deverá receber comentários e notas;
•Lista: Esta interface lista todos os comentários e deve ser acessada a partir da interface Lista Rápida. No entanto, pode-se instanciá-la juntamente com a interface de Detalhes da instância de serviço a ser comentado. Neste caso, a interface substituiria a interface Lista Rápida;
 

Observação

 

É possível comentar um conteúdo acessando diretamente a sua página de detalhes. Neste caso, o conteúdo que será exibido será o último publicado. Caso a página de detalhes acessada diretamente, nao possua conteúdo, a Lista ou Lista Rápida, nela  instanciada, não será exibida.

 

•Comentar: Esta interface permite que comentários e notas sejam postados pelos usuários que possuam a permissão "Comentar". Quando esta interface não é instanciada, ela é renderizada em um pop. Quando é instanciada em uma página, também é renderizada em um pop, com a vantagem de poder customizá-la;
•Administração: Esta interface permite que o administrador do serviço com permissão “Administrar”, controle os comentários e notas que são postados. Este controle envolve aprovação do comentário, podendo alterar a nota  e a mensagem estipulados pelo usuário;
•Resultados das Avaliações: Esta interface é disponibilizada para o administrador do serviço, acompanhar os resultados das votações dos conteúdos;
•Média das Notas: Esta interface permite que o os usuários acompanhem a votação de determinado conteúdo. Ela deve ser instanciada juntamente com a interface Lista Rápida ou Lista .

Para disponibilizar o serviço de Comentários para o usuário final, basta instanciar a interface Lista Rápida na mesma página que contém a interface de Detalhes de uma instância de serviço; instanciar a interface Lista em uma outra página que seja oculta, pois será acessada pela interface Lista Rápida; instanciar a interface "Comentar" caso queira que ela seja customizada; e por fim, instanciar a interface Média das Notas juntamente com as interfaces Lista Rápida ou Lista.

O ambiente administrativo requer uma página para a interface Administração e outra para a interface Resultados das Avaliações, em um canal restrito do portal, onde apenas o administrador do serviço terá acesso. A fim de proporcionar uma maior organização no portal, é aconselhável que estas páginas pertençam a um canal específico.

Atribuição de Permissões

O serviço Comentários trabalha com dois perfis de utilização do serviço:

•Administrar: Usuário com a permissão “Administrar” pode interferir na publicação de comentários e votos, bem como visualizar os resultados das avaliações dos conteúdos;
•Comentar: Usuário com permissão “Comentar” pode postar comentários para um determinado conteúdo, bem como estabelecer uma nota de satisfação.

Estas permissões são concedidas no controle de acesso da instância do serviço Comentários.

Na figura abaixo, há um exemplo do controle de acesso de uma instância de serviço Comentários, cujo usuário “User 1” é administradora do serviço, já que foi concedida a ela a permissão “Administrar”. Ao selecionar esta opção, automaticamente a opção “Comentar” é selecionada, pelo fato de, por definição, o administrador do serviço ter a permissão postar comentários.

 

Observação

 

O Lumis Portal oferece como sugestão de montagem deste serviço um prebuilt channel com uma estrutura de canal, páginas, e disposição das interfaces nas páginas.

Para informações sobre como instanciar canais, páginas, e atribuição de permissões no controle de acesso, consultar a seção sobre Portal Studio.

Através do right-click da interface, encontram-se disponíveis parametrizações para a instância de serviço e customizações para a interface. Consultar a seção Customização de instâncias de Interfaces.
