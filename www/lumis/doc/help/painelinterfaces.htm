Painel de Interfaces

 Top  Previous  Next

Os Painéis de Interfaces são áreas editáveis de interfaces de serviços do template de página. Podem estar localizadas em várias regiões do template determinando, portanto, a localização das interfaces a serem instanciadas nas páginas que possuam tal template aplicado.

Um template de página ao ser criado, possui como default um painel de interfaces. Fica a cargo do administrador do portal adicionar tantos painéis quanto forem necessários em diferentes áreas do template. Para isso, basta realizar o "arrastar e soltar" da opção Painel de Interfaces localizada à direita da área de edição.

A figura abaixo exibe um template de página com três painéis de interfaces e um "arraste e solte" de um painel.

Como exemplo, um template pode ter instâncias de interfaces dos serviços Busca, Login, Barra de Navegação que poderão ser replicadas em cada página do portal que possui o template aplicado. O layout definido em um template não pode ser alterado em nenhuma outra página, ou seja, nenhuma instância de interface do template removida das páginas associadas a este template. Sendo assim, utiliza-se um painel de interfaces para delimitar as áreas editáveis onde é possível adicionar e remover instâncias de interfaces nas páginas.

A figura abaixo exibe uma página associada a um template. Quando o template é aplicado em uma página, as áreas definidas no template ficam desabilitadas impedindo que nelas, instâncias de interfaces sejam removidas. Além disso, as instâncias de interfaces possuem um “T”, para distinguir as interfaces que são do template daquelas que são da página.

Observações

 

Ao aplicar template em uma página já criada e com instâncias de interfaces, o conteúdo da página preencherá, automaticamente, o painel de interface 1. Outras interfaces que forem sendo instanciadas preencherão os painéis 2 e 3 respectivamente.

 

Como o template de página deve ter ao menos um painel de interface, ao remover todos existentes de um template e salvar a edição, o portal irá ignorar todas as alterações e voltará ao último estado antes da exclusão.
