Novidades

 

•Exportação de canais:

 - possibilidade de exportação de grupos globais;

 

Melhorias
 

•Exportação:

 - opção para exportar ou não os canais filhos, através da recursividade;
 

•Importação:

 - possibilidade de importar as referências dos membros do tipo grupo ignorando as que não existem no ambiente corrente;
 

•Agora é possível posicionar interfaces em páginas com tableless, uma ao lado da outra, em uma mesma célula. Anteriormente, havia um comportamento by design na utilização do tableless, caso fosse desejado colocar duas interfaces, uma ao lado da outra. Para contornar à nível de solução, podia-se aplicar à interface uma classe (em um CSS aplicado à pagina) que posicionasse a interface de forma absoluta;

 

•Classes contendo os estilos float:left; e clear:left; foram adicionados no portal.css.

 

Alterações

 

•Alteração do JAR H2 para h2-1.0.79.jar;
 
•opção para exportar, ou não, referências a membros do tipo usuário para não estourar a memória durante a exportação ou importação;
 
•Cache criado pelo hibernate relativo a conexões externas (config/connection) passam a ter seu identificador prefixado pelo connectionId. Caso tais caches tenham sido configurados no Gerenciador de Cache do portal, será necessário refazer essas configurações para o novo identificador.
