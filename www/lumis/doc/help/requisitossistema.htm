Nesta seção estão descritos os pré-requisitos necessários para a execução do Lumis Portal.

 

Requisitos de Hardware

 

O hardware deve ser especificado em função dos requisitos de desempenho da solução que será construída com ele, sendo definido no escopo do projeto. Abaixo, segue uma sugestão de hardware mínimo para executar o Lumis Portal.

 

Servidor de Aplicação

•2 GB de memória RAM;
•20 GB de espaço livre em disco.

Servidor de Banco de Dados

•2 GB de memória RAM;
•40 GB de espaço livre em disco.

Requisitos de Software

 

Para o correto funcionamento do Lumis Portal são necessários os seguintes tipos de softwares instalados:

•       Sistemas Operacionais:

o       Microsoft Windows Server 2008 R2 e 2012;

o       Microsoft Windows 7 e 8 (só deve ser utilizado para desenvolvimento);

o       Red Hat Enterprise Linux 6 e 7;

o       CentOS 6 e 7;

o       SUSE Linux Enterprise Server 11.

•       Máquina Virtual Java:

o       Oracle Java SE 8;

o       Oracle Java SE 7 (apenas com WebLogic Server);

o       IBM J2SE 7 (apenas com Websphere Application Server).

•       Servidor de Aplicação Java:

o       Apache Tomcat 8;

o       JBoss Enterprise Application Platform 6;

o       Oracle WebLogic Server 10.3;

o       IBM Websphere Application Server 8.5. *

•       Servidor Web:

o       Nginx 1.8 e 1.9;

o       Apache HTTP Server 2.2 e 2.4.

•       Navegadores:

o       Microsoft Internet Explorer 8, 9, 10 e 11;

o       Mozilla Firefox 39 e Mozilla Firefox ESR 38;

o       Google Chrome 43.

Observação

 

Para acessar o modo administrativo do portal é necessário utilizar o Microsoft Internet Explorer 9, 10 ou 11 ou Google Chrome.

•       Banco de dados relacional:

o       Oracle 11g e 12c;

o       Microsoft SQL Server 2012 e 2014;

o       MySQL 5.5 e 5.6.

•       Repositório de Big Data:

o       Elasticsearch 1.6.0. *

 

* - Na versão 8.1 do Lumis Portal, não é possível utilizar o Elasticsearch com Websphere em decorrência de incompatibilidade entre as versões atuais destes produtos.
