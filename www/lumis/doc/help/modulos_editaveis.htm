Adicionar Módulo Editável

Um módulo editável permite alterar o conteúdo interno ao módulo diretamente pelo portal.

Para criar um novo módulo editável, basta clicar no botão Criar Módulo Editável. Para instalar um módulo editável já existente deve-se marcar a opção "Instalar módulo como editável" na interface de Deploy.

 

Funções únicas deste modelo:

•Atualizar: Permite atualizar o módulo, a atualização leva em consideração o conteúdo da versão disponível na pasta

"lumisdata/shared/source" (alterável de acordo com as configurações do servidor).

 

•Abrir no editor: Abre um editor de conteúdo com os arquivos do módulo.

 

 

 Neste editor é possível:

 - Criar, alterar, renomear  ou remover arquivos.

 - Criar, alterar, renomear ou remover diretórios.

 - Enviar arquivos.

 - Enviar arquivos em zip.

 - Baixar um diretório como zip.

 - Atualizar um diretório.

 

 Alterações feitas no código do módulo (através do editor de módulos ou quaisquer outros editores, via sistema de arquivos) não são refletidas de imediato no módulo, sendo necessário atualizar o módulo para que elas sejam, assim, aplicadas.
