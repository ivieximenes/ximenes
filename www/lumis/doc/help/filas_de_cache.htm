Filas de Cache

 Top  Previous  Next

As filas podem ser administradas através do menu Filas de Cache disponível na Administração do Portal: Configuração / Cache HTML / Filas de Cache.

 

A administração de filas permite as seguintes ações:

•Adicionar fila de publicação;
•Editar fila de publicação;
•Remover fila de publicação.

Adicionar/Editar Fila

Para adicionar uma nova fila de publicação basta selecionar a opção “Adicionar”.

Tela Adicionar Fila:

As informações necessárias para cadastrar uma nova Fila são:

•Fila: Identificador da fila. Campo obrigatório. As configurações se aplicarão a caches de página associadas a fila com este identificador;
•Número máximo de erros: numérico. Campo obrigatório. Número de erros consecutivos que faz o status de um cache de página ser alterado para: "Desatualizado com Erro";
•Decremento de prioridade por erro: numérico. Campo obrigatório. Quantidade pela qual a prioridade é reduzida quando ocorre um erro durante a geração;
•Apenas sob demanda: caso esteja marcado indica que caches pertencentes a esta fila devem ser gerados apenas se demandos por algum acesso;
•Apagar arquivos desatualizados:  Indica se arquivos relativos a caches desatualizados devem ser apagados;
•Shadow cache: Indica se Shadow Cache deve ser utilizado para caches pertencentes a esta fila;
•Isolada: Caso esteja marcado indica que os cache de páginas associadas à fila serão gerados apenas pelos threads a ela alocados (definido no Perfil do Gerador) e não utilizará threads ociosos associados a outros geradores.
•Espera por erro (s): Indica a quantidade, em segundos, de tempo a ser considerado no calculo de tempo de espera entre as gerações do cache quando ocorre um erro. O calculo do intervalo de espera é: número de erros * espera por erro.
•Incluído em operações em massa automáticas: Indica se essa fila será incluída em operações de expiração em massa.

Excluir Fila

Para excluir uma fila, é necessário selecionar a fila a ser excluída e selecionar a opção “Excluir”. Nesta opção, é possível excluir mais de uma fila por vez, selecionando as filas que se deseja excluir. O sistema exibe uma mensagem de confirmação:

Basta confirmar e a(s) fila(as) será(ão) excluída(s). A exclusão de uma fila não afeta a referência de caches a ela, apenas exclui a configuração da fila. Com isso a fila passa a utilizar as configurações padrões, detalhadas abaixo.

Configuração Padrão

Caso a fila não esteja configurada, o Lumis Portal utilizará a seguinte configuração padrão:

•Número máximo de erros: 0
•Decremento de prioridade por erro: 500
•Apenas sob demanda: Marcado
•Apagar arquivos desatualizados: Marcado
•Shadow cache: Desmarcado
•Isolada: Desmarcado
•Espera por erro (s): 60
•Incluído em operações em massa automáticas: Marcado
