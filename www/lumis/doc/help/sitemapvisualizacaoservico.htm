O serviço disponibiliza em uma página uma pré-visualização do XML do sitemap gerado. Já o arquivo pode ser acessado via url do portal.

 

Para a pré-visualização do XML do sitemap, basta clicar em Visualizar Sitemap na administração do serviço:

 

 

A pré-visualização está representada na figura abaixo, juntamente com o nome do arquivo. A pré-visualização exibe o caminho amigável de cada canal e sub-canal (não oculto) definido nas propriedades de canal e página.

 

No exemplo a seguir, existem três itens cadastrados:

•/ Portal / Website 1 - canal que é raiz de um website que responde a URL http://website1:8080/portal
•/ Portal / Website 2 - canal que é raiz de um website que responde a URL http://website2:8080/portal
•/ Portal / Intranet - canal que não pertence a nenhum website

 

O conteúdo dos sitemaps são criados sob demanda e ficam em memória:

 

 

Nessa representação, os elementos que não pertençam a nenhum website são marcados na cor vermelha. Esses elementos não aparecerão no arquivo xml.

 

Os elementos de um website diferente do website através do qual o serviço está sendo acessado serão marcados em azul. Esses elementos não aparecerão no arquivo xml, caso o mesmo seja acessado através do mesmo website sendo utilizado para acessar o serviço. Nesse exemplo, o website Website 2 é diferente do website utilizado para acessar o serviço.

 

Os elementos que estejam no mesmo website do website do qual o serviço está sendo acessado não serão marcados, sendo exibidos, assim, na cor cinza. Esses elementos aparecerão no arquivo xml caso o mesmo seja acessado através do mesmo website sendo utilizado para acessar o serviço. Nesse exemplo, o website Website 1 é o website sendo utilizado para acessar o serviço e, assim, seus elementos não estão marcados.

 

Para acessar o arquivo gerado, basta acessar a seguinte url formada por http://servidor:porta/contexto/sitemap/arquivo.xml.
