Versão 6.2.1

 Top  Previous  Next

Novidades

 

•Suporte ao servidor de aplicação JBoss Application Server 7.0.

 

 

Melhorias

 

•Disponibilização, no assistente de criação de conteúdo, de configuração que permite definir campo como uma enumeração de valores baseada em uma tabela;

 

•Otimização do acesso a informações de interfaces de um serviço.

 

Defeitos

 

•Corrigido defeito na importação de LEC atualizando conteúdo com arquivo;

 

•Corrigido defeito na importação de LEC sem preservar identificadores;

 

•Corrigido defeito na importação quando provoca remoção de instância de serviço de Integração com Redes Sociais Públicas;

 

•Corrigido defeito na importação de estrutura quando elemento foi recriado ou teve alteração de identificador amigável;

 

•Corrigido defeito na importação parcial de estrutura;

 

•Corrigido defeito na exportação de estrutura quando há cadastrado custom property com valor nulo;

 

•Corrigido defeito na exportação para ponto de restauração com nome utilizando caracteres especiais;

 

•Corrigido defeito no serviço de Fórum na postagem com caracteres acentuados ou outros específicos;

 

•Corrigido defeito do editor HTML quando em página com URL amigável;

 

•Corrigido defeito nos serviços de Sala de Chat e Atendimento Online em página com URL amigável;

 

•Corrigido defeito no serviço de Newsletter, na interface de administração de assinaturas;

 

•Corrigido defeito no Gerenciador de Website ao editar website usando idioma inglês;

 

•Corrigido defeito na geração de cache HTML quando website padrão está configurado para porta 80;

 

•Corrigido defeito no acesso à página home padrão de um website quando ela redireciona para outro website;

 

•Corrigido defeito no acesso à página de um website pela URL base de outro website;

 

•Corrigido defeito na atualização do portal a partir de versão anterior à 6.2.0 quando há página padrão com URL '*';

 

•Corrigido defeito na variável javascript g_LumisRootPath quando há website cadastrado com caminho além do contexto da aplicação web;

 

•Corrigido defeito na criação de arquivos de logs em alguns casos de ambientes Linux;

 

•Corrigido defeito na exclusão de instância de serviço em alguns ambientes com banco de dados Oracle;

 

•Ajustes na documentação de Filas de Cache HTML;

 

•Ajustes na documentação de configurações do Apache;

 

•Ajustes em textos exibidos pelo Portal Studio.

 

 

Observações

 

•Não é mais permitido usar nulo ou string vazia como valor de um custom property. Se tal valor for utilizado em um dos seguintes métodos:
olumis.portal.serviceinstance.IServiceInstanceManager.setCustomProperty(SessionConfig, String, String, String, ITransaction)
olumis.portal.serviceinstance.IServiceInstanceManager.setCustomProperties(SessionConfig, String, Map<String, String>, ITransaction)
olumis.portal.serviceinterfaceinstance.IServiceInterfaceInstanceManager.setCustomProperty(SessionConfig, String, String, String, ITransaction)
olumis.portal.serviceinterfaceinstance.IServiceInterfaceInstanceManager.setCustomProperties(SessionConfig, String, Map<String, String>, ITransaction)

a propriedade não será criada, e será removida caso exista. Durante a atualização para esta versão, todos os custom properties com valor nulo ou vazio serão removidos.
