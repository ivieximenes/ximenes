O serviço pode ser montado de diversas maneiras, dependendo de como foi arquitetado pelo administrador do portal.

Para utilizar o serviço Envie para um Amigo, basta instanciá-lo em um canal e arrastar suas interfaces para as áreas das páginas em modo de edição.

 

•Administração: Esta interface permite que o administrador do serviço com permissão “Administrar”, receba as mensagens enviadas para um “amigo”, podendo excluí-las conforme necessidade. Através desta interface também, é possível configurar o formato e disposição ;
•Detalhes da Mensagem: Esta interface é acessada a partir de uma mensagem listada na interface Administração e tem o objetivo de fornecer informações ao administrador de alguns detalhes da mensagem envidada;
•Link: Interface que disponibiliza o link “Enviar para um amigo” para exibir o formulário da mensagem com indicação a ser enviada a um amigo. Importante ressaltar que este Link está preparado para abrir uma página que contenha apenas um formulário de "Enviar Mensagem";
•Enviar Mensagem: Interface renderizada em um pop que é acessada a partir do link “Enviar para um amigo”.

Para disponibilizar o serviço para o usuário final, basta instanciar as interfaces “Link” em tantas quantas forem desejadas páginas do portal; e a interface “Enviar Mensagem” em uma única página.

O ambiente administrativo com as interfaces “Administração” e Detalhes da Mensagem” requer duas páginas em um canal restrito do portal, uma para cada interface. Além disso, a propriedade “Oculto” da página que contém esta última interface deve estar selecionada para que não seja visualizada na barra de navegação;

Somente o administrador do serviço deverá ter acesso a este canal. A fim de proporcionar uma maior organização no portal, é aconselhável que estas páginas pertençam a um canal específico, o qual está sendo chamado aqui de “Envie para um Amigo”.

 

Atribuição de Permissões

O serviço Envie para um Amigo trabalha com dois perfis de utilização do serviço:

•Administrar: Usuário com a permissão “Administrar” tem acesso a área administrativa para configuração do conteúdo e formato da mensagem a ser enviada ao “amigo” . Esta permissão engloba a permissão “Postar Mensagem”;
•Postar Mensagem: Usuário com esta permissão pode recomendar a página corrente ao “amigo”.

Estas permissões são concedidas no controle de acesso da instância do serviço Envie para um Amigo.

Na figura abaixo, há um exemplo do controle de acesso de uma instância de serviço, cujo usuário “Lívia” tem a permissão de “Postar Mensagem”. Portanto, para ela estará disponível na página visualizada, o link “Envie para um amigo”.


 

Observação

 

O Lumis Portal oferece como sugestão de montagem deste serviço um prebuilt channel com uma estrutura de canal, páginas, e disposição das interfaces nas páginas.

Para informações sobre como instanciar canais, páginas e atribuição de permissões no controle de acesso, consultar a seção sobre Portal Studio .

Através do right-click da interface, encontram-se disponíveis parametrizações para a instância de serviço e customizações para a interface. Consultar a seção Customização de instâncias de Interfaces.
