Dependência entre componentes

 Top  Previous  Next

Um componente pode definir dependência com outros componentes, inclusive dependência com uma versão específica de um componente. Essa dependência é feita utilizando-se o nó constraints do arquivo de definiação do componente.

 

A dependência pode ser:

•De existência: valida somente se um determinado está instalado.
•De versão igual a: valida se um componente está instalado exatamente na versão requerida.
•De versão igual a ou superior: valida se um componente está instalado na versão requerida ou em uma versão posterior.
