No Gerenciador de Usuários é possível visualizar todos os usuários cadastrados e administrar o cadastro dos mesmos.

A interface administrativa lista  informações gerenciais tais como Login do usuário e Último login do mesmo.

A administração envolve as seguintes ações:

 

•Adicionar: Permite adicionar um novo usuário no portal. Os seguintes campos podem ser preenchidos no formulário:

 

oImagem do perfil: imagem do usuário.
oNome: primeiro nome;
oNome do meio: sobrenome ou nome do meio;
oÚltimo nome: último nome do usuário.

Ex.: João Matias da Silva – o nome é João, o nome do meio é Matias e o último nome da Silva.

oE-mail: endereço de e-mail do usuário;
oLogin: login ou nome de usuário;
oSenha: senha para autenticação;
oConfirmação de senha: digitar novamente a senha;
oDesabilitado: permite habilitar e desabilitar o acesso do usuário ao portal;
oMultisessão: permite habilitar e desabilitar a permissão ao usuário de abrir várias sessões ao mesmo tempo.
•Editar: Permite editar as propriedades de usuários globais e locais já cadastrados;
•Excluir: Permite excluir um usuário selecionado na lista. Usuários locais são automaticamente excluídos do gerenciador de uusários, quando o canal em que eles foram criados, for excluído;
•Membro de: Permite adicionar o usuário selecionado em outros os grupos globais e locais que o administrador tenha acesso. Por default, o usuário faz parte dos grupos “Todos os usuários” e “Usuários cadastrados”. Exceto o usuário "Guest".

 

É possível visualizar também se tal grupo pertence a um outro indiretamente, por transitividade.

O portal possui os seguintes usuários padrão:

•Guest : usuário de sistema e que pertence ao grupo padrão "Todos os usuários". É utilizado pelo portal para acesso anônimo;
•LumisFrameworkClockUser: O serviço interno de clock utiliza este usuário para fazer monitoração de eventos relacionados a datas. Como por exemplo: publicar conteúdos no portal quando a data e hora de publicação forem atingidas. Este usuário programático pertence aos grupos padrão do portal "Administradores do Portal", "Usuários cadastrados" e "Todos os usuários";
•LumisPageCacheUser: O serviço interno de cache utiliza  este usuário para gerar a página HTM de uma página configurada para ser cacheada. Porém, é necessário que este usuário tenha a permissão de visualização da página e da instância de serviço. Este usuário programático pertence aos grupos padrão do portal "Administradores do Portal", "Usuários cadastrados" e "Todos os usuários";
•System : usuário de sistema e que pertence ao grupo padrão "Todos os usuários" e "Usuários cadastrados". É utilizado pelo portal para acesso determinadas informações de sistemas.
