MIME Types

Os arquivos cadastrados nos repositórios do portal, tanto documentos quanto imagens, possuem ícones dos seus aplicativos de origem, que estão registrados no portal. Novos mime types podem ser cadastrados nesta seção para que possam ser reconhecidos pelo portal.

Para adicionar um novo mime type deve-se acionar “Adicionar” e cadastrar as seguintes informações:

 

oMime Type: o mime deve ser cadastrado corretamente de acordo com o aplicativo desejado, para que o portal o reconheça e possa disponibilizar o ícone. No endereço: http://www.mimetype.org/ há um lista de mime types;
oÍcone: Deve-se informar o caminho relativo a www onde está localizada a imagem do ícone.

Estas informações básicas podem ser alteradas, selecionando-se o mime type e em seguida, "Editar". Todos os mimes cadastrados, bem como os distribuídos com o Lumis podem ser excluídos. Dessa forma, os arquivos localizados no repositório do portal, ficarão sem o ícone correspondente ao mime excluído.
