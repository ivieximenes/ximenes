Gerenciador de Módulos

O Lumis Portal oferece um serviço para instalação de módulos. Cada módulo pode conter classes, serviços, temas, componentes e contextos de negócio que serão disponibilizados no Portal.

É possível realizar o deployment de tais módulos através do gerenciador de módulos de deployment do Lumis.

 

O serviço pode ser encontrado em Configuração -> Deployment e Performance Tuning -> Módulos.

 

 

Serviço Gerenciador de Módulos:

 

Com o serviço Gerenciador de Módulos é possível realizar as seguintes operações:

•Criar um Módulo Editável;
•Deploy;
•Baixar um módulo.
•Desinstalar um módulo;
•Visualizar os objetos de um módulo;
•Verificar mensagens de erro da última operação realizada em um módulo.

 

Baixar um Módulo

Para baixar um módulo é necessário escolher um módulo e acionar a opção “Baixar arquivo”. Nesta opção, não é possível baixar mais de um módulo por vez. Obs: Módulos são baixados compactados em .zip.

Desinstalar Módulo

Para desinstalar um módulo é necessário selecionar o módulo e selecionar a opção “Desinstalar”. Nesta opção, é possível desinstalar mais de um módulo por vez, selecionando os itens que se deseja desinstalar. Ao fazer isso, o usuário é levado à tela de confirmação de deployment, que listará todas as ações a serem tomadas.

Verificar mensagens de erro

Caso o módulo não tenha sido desinstalado, é possível consultar as mensagens de erro clicando sobre o número de erros do módulo:

Além disso, os erros podem ser consultados no log.

Para informações sobre como criar módulos para deployment, consultar Módulos, em Customização e Desenvolvimento.

Para informações técnicas sobre o deployment do Lumis, consultar o javadoc lumis.portal.deployment.
