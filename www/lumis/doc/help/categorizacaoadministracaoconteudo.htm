O serviço Categorização possui um ambiente administrativo disponibilizado pelo administrador do portal, para o cadastro dos termos pelo editor responsável. Uma vez o conteúdo cadastrado, é possível reeditá-lo ou excluí-lo. Para cadastrá-lo, basta acionar “Adicionar” e preencher as seguintes informações:


 

•Termo: Campo obrigatório que deve ser preenchido com o nome do termo;
•Descrição: Campo de preenchimento facultativo que pode ser preenchido com uma breve descrição;
•Termo pai: Campo de preenchimento facultativo que pode-se determinar a hierarquia do termo, ou seja, qual termo é seu pai.

É possível associar a outros termos de outros idiomas para a publicação multilíngue. Termos associados a outros termos para publicação multilíngue sempre terão o mesmo termo pai. Ao adicionar ou editar um termo associado, o termo pai selecionado será aplicado a todos os demais termos associados a este para publicação multilíngue.
 

Cada termo cadastrado fica disponível na lista administrativa para possíveis manutenções de informações.
 

Observação

 

Para informações publicação multilingue do conteúdo e visualização do conteúdo, consultar a seção “Funcionalidades Gerais dos Serviços de Gestão de Conteúdo”.
