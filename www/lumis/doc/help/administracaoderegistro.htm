O serviço de Produtor WSRP possui interfaces específicas para auxiliar na gerencia dos registros.  As interfaces são a Gerenciar Registros e os Detalhes do Registro.  Ambas são interfaces que podem ser instanciadas em qualquer página, porém, como dito no tópico de Montagem, deveriam ser colocados em páginas próprias dentro de uma área controlada onde somente o  administradores/montadores do portal possuam acesso.

 

Gerenciar Registros

A interface de gerenciar registros permite que se adicione um novo registro outband, se atualize o lifetime ou exclua um registro.  Na lista de registros, são exibidos todos os registros, inclusive os que foram criados inband.  Desta interface é possível ver os detalhes de um consumidor cliando em seu link.  O link é gerado automaticamente caso exista uma interface de Detalhes do Registro instanciada.
 

 

Adicionar Registro Outband

1) Na interface de Gerenciar Registros, pressione o botão Adicionar Registro Outband.

2) A interface para inserir um novo registro outband será exibida.
 


 

3) Preencha as informações solicitadas e aperte no botão ok.

 

Extender Validade (Lifetime)

1) Na interface de Gerenciar Registros, pressione o botão Extender Validade.

2) Preencha a validade com o novo valor e aperte o botão ok.
 

Observação:

O Lumis Portal como produtor WSRP não suporta extensão de lifetime utilizando a API do WSRP.  Para realizar a extensão de lifetime o consumidor de um serviço WSRP de Portal Lumis deverá solicitar ao administrador uma extensão do registro.

 

Excluir Registro

1) Na interface de Gerenciar Registros, selecione um registro e aperte o botão Excluir.  A remoção do registro impossibilita que o consumidor continue utilizando o Lumis Portal como produtor para aquela instancia do serviço.

 

Detalhes do Registro

1) Na interface de Gerenciar Registros, clique num registro.

2) A interface de Detalhes do Registro será exibida contendo suas informações.  A tela de Detalhes do Registro é uma tela apenas de exibição, não havendo qualquer interação para alteração do estado do portal.
