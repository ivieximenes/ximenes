Montagem

O serviço pode ser montado de diversas maneiras, dependendo de como foi arquitetado pelo administrador do portal.

Para utilizar o serviço HTML, basta instanciá-lo em um canal e arrastar sua interface para as áreas das páginas em modo de edição.

•Exibe HTML: Esta interface permite exibir o conteúdo HTML em forma de texto ou imagem.

 

Atribuição de Permissões

Para que o usuário realize upload de arquivos, é necessário que ele tenha a permissão de Efetuar Upload de arquivo no Gerenciador de acesso do portal.

Parametrização da Instância de Serviço

Através do right-click na interface Exibe HTML são exibidos dois editores: Editor HTML e Editor Texto.

•Editor HTML: Permite a edição do conteúdo HTML através de uma interface com usuário rica;

•Editor de Texto: Permite a edição do código-fonte do conteúdo HTML em um editor de texto.

 

O Administrador do Portal pode definir para ambas opções onde os dados serão salvos: na Instância de Serviço ou na Instância de Interface do Serviço.

Se os dados forem salvos na Instância de Interface do Serviço, eles são específicos da instância de interface sendo editada e não serão compartilhados com outras instâncias de interface.

Se os dados forem salvos na Instância de Serviço, eles serão compartilhados entre as instâncias de interfaces desta mesma instância de serviço. Neste caso, qualquer instância de interface de Exibe HTML, pertencente à mesma instância de serviço, que não possua dados salvos no escopo de Instância de Interface do Serviço, utilizará estes mesmos dados.

 

Observações

 

Para informações sobre utilização do editor HTML, consultar a seção sobre Funcionalidades Gerais dos Serviços de Gestão de Conteúdo.

Para informações sobre como instanciar canais e páginas, consultar a seção sobre Portal Studio.

Através do right-click da interface, encontram-se disponíveis parametrizações para a instância de serviço e customizações para a interface. Consultar a seção Customização de instâncias de Interfaces. Entre elas o repositório de imagens a ser utilizado no editor HTML.
