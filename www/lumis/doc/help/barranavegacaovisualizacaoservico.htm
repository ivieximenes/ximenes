Visualização do Serviço

O serviço disponibiliza alguns layouts de menus para a navegação no portal.

Os menus, geralmente, são as barras de navegações que representam a hierarquia de canais e páginas (não ocultos) que compõem o portal.

A imagem abaixo exibe o menu na sua forma hierárquica vertical:

Ao clicar em item de menu, o usuário do portal é remetido para uma página, mesmo no caso do item de menu ser um canal. Neste caso, o usuário é remetido para a primeira página do canal.

À medida que o usuário navega pelas páginas, o serviço exibe o caminho das seções navegadas, até a página corrente. Quando a página corrente é uma página de detalhes (e esta está configurada como oculta), é exibido o título do conteúdo que a página de detalhes renderiza, como é exemplificado na figura abaixo:

Geralmente, o Caminho da página fica localizado na parte superior esquerda da página, e é linkável para cada página das seções que o compõe.
 

Observações

 

 

Banners do tipo flash podem sobrepor algumas renderizações, como barra de menu com estilo horizontal. Isso ocorre porque o plugin do flash tende a colocá-lo na frente da renderização da página. Existem configurações específicas que devem ser feitas no arquivo em flash para que ele tenha um comportamento correto.

 

É necessário que o usuário tenha permissão de visualização da instância de serviço Barra de Navegação para visualizar o menu. Além disso, os itens de menu que referenciam canais e páginas na barra de navegação são exibidos para o usuário conforme suas permissões de visualização respectivamente para o canal e página. Porém, os usuários e/ou grupos somente visualizarão o item se este estiver publicado para tais.
