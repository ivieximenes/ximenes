Visualização do Serviço

 Top  Previous  Next

Duas visualizações distintas são possíveis nesse serviço. A primeira é a visualização de dados públicos, que exibe um subconjunto dos dados do perfil social do usuário. A segunda é a visualização de dados completos, que apresentam todos os dados do perfil do usuário.

 

Perfil público

O perfil público de um usuário consiste em:

•Nome
•Descrição
•Foto
•Título
•Departamento
•Língua
•Código de área
•Telefone
•Email

O perfil completo de um usuário consiste em todos os dados de seu perfil social.

Observações
1.As interfaces de visualização desse serviço esperam receber um parâmetro chamado lumUserId para filtrar o perfil de usuário a ser exibido. Caso esse parâmetro não esteja presente, o perfil do usuário atual será exibido.
2.Caso o usuário a ter seu perfil exibido ainda não possua perfil criado, um perfil mínimo será exibido, contendo apenas seu nome de usuário de seu cadastro no Lumis Portal (composto por nome, nome do meio caso exista e sobrenome caso exista).
3.O perfil do usuário será carregado de acordo com a instância do serviço e da língua de navegação do usuário corrente.
