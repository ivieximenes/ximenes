O serviço pode ser montado de diversas maneiras, dependendo de como foi arquitetado pelo administrador do portal.

Para utilizar o serviço Fale Conosco, basta instanciá-lo em um canal e arrastar suas interfaces para as áreas das páginas em modo de edição.

 

•Administração: Esta interface permite que o administrador do serviço com permissão “Administrar”, recebas as mensagens por e-mail, além de visualizá-las e gerenciá-las na área administrativa;
•Administração de Categorias: Esta interface permite que o administrador do serviço, com permissão "Administrar" cadastre as possíveis categorias que serão selecionadas pelo usuário final para classificar uma mensagem;
•Fale Conosco: Esta interface permite o usuário com permissão “Postar Mensagem” cadastre um formulário e o envie logo em seguida.

Para disponibilizar o serviço de Fale Conosco para o usuário final, basta instanciar a interface “Fale Conosco” em uma página do Portal.

O ambiente administrativo requer duas páginas, uma para a interface Administração, e uma para a interface Administração de Categorias , em um canal restrito do portal, onde apenas o administrador do serviço terá acesso, após o login. A fim de proporcionar uma maior organização no portal, é aconselhável que esta página pertença a um canal específico, o qual está sendo chamado aqui de “Fale Conosco”.

 

 

 

Atribuição de Permissões

O serviço Fale Conosco trabalha com dois perfis de utilização do serviço:

•Administrar: Usuário com a permissão “Administrar”, recebe as mensagens por e-mail e tem acesso à área administrativa para visualização e controle das mesmas, e para cadastro das categorias”. esta permissão engloba "Postar Mensagem";
•Postar Mensagem: Usuário com permissão “Postar Mensagem” pode cadastrar o formulário e submetê-lo em seguida.

Estas permissões são concedidas no controle de acesso da instância do serviço Fale Conosco.
