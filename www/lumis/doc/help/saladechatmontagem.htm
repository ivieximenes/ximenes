O serviço pode ser montado de diversas maneiras, para o usuário com perfil de administrador do serviço e para usuários com perfis de moderador e participante, dependendo de como foi arquitetado pelo administrador do portal.

Para utilizar o serviço Sala de Chat, basta instanciá-lo em um canal e arrastar suas interfaces para as áreas das páginas em modo de edição.

 

•Administração: Esta interface permite o cadastro e manutenção de salas de chat pelo administrador do serviço. Nesta interface é possível definir por quanto tempo a sala estará publicada no portal, por quanto tempo estará aberta ou fechada e se será moderada ou não;
•Lista Rápida: Exibe salas (abertas ou com Estado= Aguardando) mais recentemente cadastradas, que estão no período de publicação; e horário de início e fim. Possui um link “mais” para uma lista completa delas;
•Lista: Exibe todas as salas cadastradas (abertas ou com Estado= Aguardando), respeitando-se também período de publicação; e horário de início e fim;
•Chats realizados: Exibe uma lista de salas que estão fechadas e onde cada uma delas podem ser consultadas para visualização das mensagens trocadas entre os participantes;
•Detalhes: Exibe as informações da sala de chat. O usuário final tem acesso a essa interface ao selecionar uma sala na Lista ou Lista Rápida;
•Participar: Esta interface é exibida quando o usuário seleciona uam sala aberta disponível na Lista ou Lista Rápida. Permite que seja informado um apelido, quando uma sala for selecionada na Lista ou Lista Rápida. Deve ser instanciada juntamente com a interface Detalhes;
•Histórico da sala de chat: Esta interface é exibida quando uma sala é selecionada na lista de Chats realizados. Exibe todas as mensagens públicas trocadas entre os participantes de determinada sala. As mensagens reservadas somente são exibidas se o usuário que selecionou a sala foi o emissor ou receptor de tais mensagens. Também deve ser instanciada juntamente com a interface Detalhes;
•Sala de Chat: Esta interface exibe a sessão do usuário participante, ao acionar “participar” na interface “Informações sobre a sala de chat”. Permite enviar e visualizar mensagens que são postadas em uma sala. Esta interface é renderizada em um pop;
•Sala de Chat - moderadores: Esta interface exibe a sessão do usuário moderador, ao acionar “participar” na interface “Informações sobre a sala de chat”. Permite enviar e visualizar mensagens que são postadas em uma sala. Além disso, permite ao moderador aprovar ou não as mensagens que são postadas antes que elas sejam visualizadas pelos destinatários. Esta interface é renderizada em um pop.

Essas interfaces são essenciais para o funcionamento do serviço.

Para disponibilizar o serviço Sala de Chat para moderadores e participantes, basta instanciar em páginas diferentes do portal, as interfaces Lista Rápida, Lista, Detalhes, Sala de chat e Sala de Chat – moderadores. As páginas que possuírem essas três últimas interfaces devem ter sua propriedade “Oculto” selecionada para não serem visualizada na barra de navegação. Juntamente com a interface Detalhes devem ser instanciadas as interfaces Participar e Histórico da sala de chat.

O ambiente administrativo do serviço requer apenas uma página com a interface Administração, em um canal restrito do portal, onde apenas o administrador do serviço terá acesso, após o login. A fim de proporcionar uma maior organização no portal, é aconselhável que esta página pertença a um canal específico, o qual está sendo chamado aqui de “Sala de Chat”.

 

Atribuição de Permissões

O serviço Sala de Chat trabalha com diferentes perfis de utilização do serviço:

•Administrador: Usuário com a permissão “Administrar”, pode cadastrar salas de chat.  Esta permissão inclui também as permissões “Moderar” e “Participar”;
•Moderador: Usuário com permissão “Moderar”, é um candidato a moderador de uma sala de chat. Possui uma sessão especial da sala onde é moderador, podendo intervir nas mensagens que serão visualizadas;
•Participante: Usuário com permissão “Participar” pode enviar e visualizar as mensagens públicas; e enviar mensagens reservadas e visualizar aquelas em que ele foi ou emissor ou receptor.

Estas permissões são concedidas no controle de acesso da instância do serviço. Na figura abaixo, há um exemplo do controle de acesso de uma instância de serviço, cujo usuário “User 2” está com permissão “Moderar” e por conseqüência “Participar”.
