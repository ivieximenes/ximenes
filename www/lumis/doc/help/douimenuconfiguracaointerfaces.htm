Menu de Configuração de Interfaces

 Top  Previous  Next

Uma das vantagens do desenvolvimento de aplicações sobre o doui framework é a possibilidade de criar aplicações genéricas e parametrizáveis para que possam ser reutilizadas em cenários distintos.

Serviços criados a partir do DOUI possuem geralmente no right-click sobre as interfaces administração e lista os seguintes itens de menu: Estilo, Números de Itens, Campos, Ordenação, Filtros, Repositório de Imagens e Repositório de Documentos. Estas opções são típicas para manipulação de dados.

Serviços do Portal que não utilizam o DOUI possuem no right-click apenas as opções Propriedades e Atualizar pelo fato de não manipularem dados.

Para acrescentar outros itens de menu, deve-se implementar uma nova classe para renderizar o pop-up. Por exemplo, o código abaixo do servicedefinition.xml do serviço HTML, implementa um item de menu “Editor HTML” para a interface do serviço HTML.

<interface id="display" name="Display" className="lumis.service.html.DisplayHtml" >

  <menu>

    <item name="Editor HTML" type="addInterface">

      <interfaceId>lumis.service.html.edit</interfaceId>

    </item>

  </menu>

</interface>

No Lumis Portal Administration é possível acessar configurações em uma interface através do right-click sobre ela, onde são exibidos vários itens de menu.
