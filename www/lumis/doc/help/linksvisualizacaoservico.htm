Visualização do Serviço

Para o usuário final da solução, é disponibilizada uma lista de links, que pode ser uma lista rápida ou completa deles. A lista rápida contém os links mais recentemente cadastrados, além de um link “mais” que redireciona para uma lista completa deles. Na lista completa, por sua vez, todos os links cadastrados são agrupados por categorias de assuntos.

Nas figuras abaixo, tem-se ao todo quatro links e são tanto exibidos na lista rápida quanto na lista completa deles. A diferença é que na lista completa eles estão agrupados por categorias ou assuntos (“Esporte”, “Política” e “Vestibular 2006”).

O link pode direcionar para um conteúdo interno, para um documento do portal, para uma página interna ou externa.

Caso a página interna, para qual o link está apontando, tenha sido removida do portal, o link redirecionará para a página de erro 404 customizada.

Caso o documento, para qual o link está apontando, tenha sido removido do repositório, o link redirecionará para a home do portal, caso o usuário tenha acesso.

 

Observação

 

Além do usuário necessitar da permissão de visualização da instância de serviço para poder visualizar a lista de links, é necessário que ele pertença à lista "Publicação para Usuários e Grupos" de determinado link para poder visualizá-lo na lista.

Além disso, quando o contéudo de um link é uma página do portal ou um documento do repositório de documentos, é necessário que o usuário tenha permissões de visualização de tal página, da instância de serviço que está localizada na página e da instância de serviço de documentos.
