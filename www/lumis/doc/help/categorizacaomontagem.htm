Montagem

O serviço pode ser montado de diversas maneiras, para os editores de conteúdo e para o usuário final da solução, dependendo de como foi arquitetado pelo administrador do portal.

Para utilizar o serviço de Categorização, basta instanciá-lo em um canal e arrastar suas interfaces para as áreas das páginas em modo de edição. Suas interfaces correspondem a um ambiente administrativo de cadastro do termo e áreas de visualização do serviço pelo usuário final.

 

•Administração: Esta interface permite o cadastro e manutenção do termo  pelo editor de conteúdo;
•Lista: Esta interface disponibiliza a lista de termos cadastrados;
•Detalhes: Esta interface exibe os detalhes do termo selecionado a partir da  interface “Lista”. Exibe também a lista de conteúdos que estão categorizados com termo determinado;
•Termos: Esta interface permite listar todos os termos a que um conteúdo está relacionado.

Para disponibilizar o serviço de Categorização para o usuário final, basta instanciar as interfaces Lista e Detalhes em páginas diferentes do canal. A página que contém a interface Detalhes deve ter a propriedade “Oculto” selecionada, pois esta página será acessível a partir das páginas correspondentes às interfaces Administração e Lista, e até mesmo Busca com Resultados (do serviço de Busca”). Já a interface Termos deve estar instanciada juntamente com a interface de detalhes da instância de serviço cujo conteúdo deseja-se que seja categorizado (como foi utilizado na ilustração, o serviço de Avisos).

O ambiente administrativo de termos requer apenas uma página em um canal restrito do portal, onde apenas os editores de conteúdo terão acesso, após o login. A fim de proporcionar uma maior organização no portal, é aconselhável que esta página pertença a um canal específico, o qual está sendo chamado aqui de “Categorização”.

 

Atribuição de Permissões

Para que o editor de conteúdo possa acessar o ambiente administrativo, o administrador do portal deve atribuir a ele a permissão “Editar conteúdo de instância de serviço”. Esta permissão é concedida no controle de acesso da instância do serviço.

 

Observação

 

O Lumis Portal oferece como sugestão de montagem deste serviço um prebuilt channel com uma estrutura de canal, páginas, e disposição das interfaces nas páginas.

Para informações sobre como instanciar canais, páginas e atribuição de permissões no controle de acesso, consultar a seção sobre Portal Studio .

Através do right-click da interface, encontram-se disponíveis parametrizações para a instância de serviço e customizações para a interface. Consultar a seção Customização de instâncias de Interfaces.
