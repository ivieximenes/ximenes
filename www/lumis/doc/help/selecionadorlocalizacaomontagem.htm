Montagem

O serviço pode ser montado de diversas maneiras, dependendo de como foi arquitetado pelo administrador do portal.

Para utilizar o serviço Selecionador de Localização, basta instanciá-lo em um canal e arrastar suas interfaces para as áreas das páginas em modo de edição.

•Administração:  Esta interface permite o cadastro e manutenções (alterações e exclusões) dos idiomas , pelo administrador do serviço;
•Administração de Expressões: Esta interface permite que expressões e objetos das soluções do portal sejam traduzidos para diversos idiomas (de acordo com os idiomas cadastrados);
•Selecionador de Localização: Esta interface pode ser instanciada em várias páginas (ou em um template de página para ser aplicada em várias páginas) para que o usuário possa alterar o idioma do portal.
 

Observação

 

Para informações sobre como instanciar canais e páginas, consultar a seção sobre Portal Studio .

Através do right-click da interface, encontram-se disponíveis parametrizações para a instância de serviço e customizações para a interface. Consultar a seção Customização de instâncias de Interfaces.

O idioma default do portal está configurado no arquivo /lumisdata/config/lumisportalconfig.xml. Maiores informações em Configurações do Portal.

No XSL padrão do produto, navegadores com javascript desabilitado poderão perder parâmetros de outras interfaces ao utilizar o link de troca de idioma. Se o XSL for customizado, apenas executando a operação através do javascript onclick fornecido, manterá os parâmetros de outras interfaces. Navegando pelo link acessível fornecido, o idioma trocará, mas outras interfaces poderão perder parâmetros.
