Parametrização da Instância de Serviço

Pode-se configurar para qual domínio os grupos serão cadastrados. Para isso, o administrador deve selecionar via right-click qualquer interface do serviço de gerenciador de grupos e informar o domínio.

Ao selecionar a opção domínio, é disponibilizada uma lista de domínios, um deles o domínio "global" e os outros (se houver), domínios locais representados pelos prefixos dos canais que possuem grupos locais.

 

Se o administrador determinar que o domínio será "global", o cadastro será de grupos considerados globais ao portal. Caso o administrador escolha algum domínio local, o cadastro de grupos será apenas para determinado canal representado por aquele domínio.

 

Se o administrador determinar que o domínio será "Todos", o cadastro será de grupos considerados globais ao portal, porém, a interface administrativa exibirá tanto grupos globais quanto grupos locais.

Atribuição de Permissões

Para que um usuário possa cadastrar grupos globais, ele deve ter a permissão "Gerenciar Grupo" no controle de acesso global. Este controle de acesso é aquele acessado via right-click sobre qualquer canal, na opção "Gerenciar Acesso do Portal".
