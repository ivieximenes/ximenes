Sincronização com Repositórios Externos

 Top  Previous  Next

O Lumis Portal permite a sincronização com repositórios externos de grupos e usuários, através do protocolo LDAP (Lightweight Directory Access Protocol), permitindo que, após a importação, usuários acessem a solução.

 

A importação pode ser manual ou automática (de acordo com configuração na administração do portal) e é feita em duas fases. A primeira fase lê os dados do repositório externo e cria um arquivo XML com os dados dos grupos e usuários. A segunda lê este arquivo XML e grava as informações no banco de dados da solução Lumis.

Esta versão traz apenas a importação do Active Directory da Microsoft, mas outros módulos de leitora podem ser desenvolvidos para acessar outros repositórios diferentes.

Toda a administração da base de usuários e grupos é feita no repositório de origem. Os usuários sincronizados podem ou não ser autenticados pelo portal.

O módulo que acessa o AD não salva as senhas dos usuários, a autenticação deve ser configurada no JAAS. Os grupos também são importados com seus respectivos membros.

 

Observação

 

Para maiores informações consultar a seção sobre Importação de Usuários e Grupos.
