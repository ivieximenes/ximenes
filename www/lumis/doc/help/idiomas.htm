Idiomas

 Top  Previous  Next

Nessa aba são apresentadas as configurações relativas ao idioma do canal sendo alterado.

 

 

Idiomas suportados:

Esta opção faz com que o Lumis Portal só permita o suporte a determinados idiomas no contexto do canal editado.

 

Campos:

•Herdar idiomas suportados: utilza os idiomas suportados do canal pai. Caso o canal já seja o canal raiz (ou seja, não possui canal pai), os idiomas suportados serão todos os idiomas cadastrados no portal(1).

 

•Definir lista de idiomas  suportados: define uma lista de idiomas suportados(1).

 

Detecção de idioma do browser do cliente:

Esta opção ativa a detecção do idioma através da requisição do navegador do cliente. Para a detecção funcionar adequadamente, o navegador do usuário deve encaminhar o cabeçalho HTTP "Accept-Language" de acordo com a especificação HTTP 1.1(2). O Lumis Portal irá identificar automaticamente o idioma atual e se este for um dos idiomas suportados pelo canal, o utilizará.

 

Campos:

•Herdar detecção de idioma: utiliza a detecção de idioma do canal pai. Caso o canal já seja o canal raiz (ou seja, não possui canal pai), a detecção do idioma será ativa.

 

•Detecção de idioma do browser do cliente: ativa ou desativa a detecção de idioma através do navegador do cliente.

 

Idioma padrão:

Esta opção permite selecionar qual dos idiomas será considerado o idioma padrão do canal selecionado.

 

Campos:

•Herdar idiomas padrão: utiliza o idioma padrão do canal pai. Caso o canal já seja o canal raiz (ou seja, não possui canal pai), o idioma padrão será um dos idiomas suportados.

 

•Idioma padrão: define o idioma padrão da área do portal delimitada pelo canal sendo editado.

 

Observações

 

(1) Os idiomas cadastrados no portal podem ser alterados no serviço Selecionador de localizações.

(2) http://www.w3.org/Protocols/rfc2616/rfc2616-sec14.html#sec14.4
