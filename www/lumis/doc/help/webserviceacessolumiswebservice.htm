Acesso ao Lumis Web Service

 Top  Previous  Next

As plataformas de desenvolvimento mais recentes, tais como Microsoft .Net e Java, possuem ferramentas que analisam o arquivo WSDL e criam automaticamente classes de acesso a cada web service (também chamadas de classes stub ou proxy). Essas classes encapsulam a complexidade da conexão HTTP e a codificação dos dados em XML e SOAP, possibilitando ao desenvolvedor da aplicação cliente, utilizar os métodos da API do Lumis, acessando-os através de uma classe local.
