Visualizar Entradas de Auditoria

O relatório de Auditoria permite visualizar as entradas de auditoria salvas no banco de dados através da interface do Portal. Na seção “Configurações / Informações / Auditoria / Visualizar Entradas de Auditoria” é possível listar as entradas de Auditoria:

O serviço de visualização das entradas de auditoria permite filtrar as entradas pelos campos:  "Categoria", "Tipo", "Descrição", "Login do usuário" e pelo período em que a entrada foi registrada. Os dados exibidos para as entradas de auditoria são: categoria, tipo, título, data e login do usuário.
