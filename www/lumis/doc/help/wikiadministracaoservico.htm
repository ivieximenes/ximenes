Administração do Serviço

O usuário com a permissão de administração da instância do serviço pode excluir os artigos quando necessário.

 

 

Quando um artigo excluído está sendo referenciado em outro artigo, o link para ele é direcionado para a página de erro 404.

 

O artigo principal, aquele que inicialmente é o main nunca é excluído, isso porque quando a página de exibição do wiki é acessada, tal artigo é exibido.
