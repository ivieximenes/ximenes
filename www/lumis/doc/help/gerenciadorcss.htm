Gerenciador de CSS

 Top  Previous  Next

Arquivos CSS's customizados com formatações para o portal, canais e páginas devem ser registrados para que o portal possa incorporá-los, antes de serem efetivamente utilizados. Os arquivos customizados devem estar em “www”, ou em subdiretórios, para que sejam interpretados pelo portal.

Através do gerenciador de CSS é possível:
 

•Adicionar: permite registrar arquivo CSS já implementado;

oNome: nome de identificação do CSS quando for utilizado para aplicação em canais e páginas;
oDescrição: breve descrição sobre o objetivo do CSS;
oCaminho do arquivo: caminho do arquivo CSS  que deve ser relativo ao diretório www.Se forem utilizadas contra-barras “\” no caminho do arquivo, apos a confirmação do registro do CSS , automaticamente serão trocadas por barras “/”.
 
•Editar: permite editar as propriedades do arquivo CSS selecionado na lista, como a descrição do conteúdo do arquivo, o nome representativo, e o próprio nome do arquivo. Quaisquer alterações no conteúdo do arquivo não impactam no registro do CSS no portal, ou seja, automaticamente  as alterações serão adotadas, a não ser que ocorra cache de browser;
 
•Excluir: permite excluir o arquivo CSS selecionado na lista. A exclusão não afeta o arquivo fisicamente, apenas o seu registro no portal. Desta forma, a solução deixa de utilizar as formatações definidas no arquivo CSS excluído e passa a utilizar as configurações default definidas no arquivo portal.css e em outros CSS registrados.
