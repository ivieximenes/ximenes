Neste tópico é exemplificado um serviço utilizando um dataProvider em JavaScript.

O dataProvider preenche o source da instancia de serviço com o nome e descrição do serviço, um campo com informações adicionais, informações do usuário atual: login, primeiro nome, email e data do ultimo login e grupos aos quais ele pertence.

 

Observações

Os scripts em JavaScript tem algumas limitações. Não é possível por exemplo, acessar os atributos e métodos das classes Java extendidas por um script JavaScript.
