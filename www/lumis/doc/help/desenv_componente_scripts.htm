Scripts de instalação, atualização e desinstalação

 Top  Previous  Next

Um componente pode possuir scripts que atuam como scripts de instalação, atualização e desinstalação.

Esses scripts devem estar localizados em uma pasta chamada componentfiles/setup, que deve ser localizada dentro da pasta que contém o arquivo de definição do componente.

 

Os arquivos de script de instalação, atualização e desinstalação devem se chamar, respectivamente:

•component-install-script.xml
•component-upgrade-script.xml
•component-uninstall-script.xml
