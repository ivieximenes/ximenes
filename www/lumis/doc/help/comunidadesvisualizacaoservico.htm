Para o usuário final da solução, são disponibilizadas uma lista contendo todas as comunidades públicas ou restritas, e uma lista contendo todas as comunidades das quais o usuário participa. Na lista de comunidades do usuário, dependendo das permissões do usuário, também é disponibilizado um botão para adicionar novas comunidades.

Lista de todas as comunidades

       

Lista de comunidades em que o usuário participa

Em ambas as listas, ao clicar no ícone ou no título da comunidade, caso o usuário não faça parte dessa comunidade, ele é direcionado para uma página de detalhes da comunidade, onde, além do ícone, do título e da descrição da comunidade, é informado o nível de privacidade dessa comunidade e é disponibilizado um botão para participar da comunidade, ou requisitar a participação, caso seja uma comunidade restrita.

Detalhes de uma comunidade restrita

Caso o usuário faça parte dessa comunidade, ele é direcionado para a primeira página do canal da comunidade, no caso do pre-built channel "Mural", que contém um cabeçalho com o título, o ícone e a descrição da comunidade, um menu para navegar pelas áreas da comunidade e uma lista de NewsFeeds, onde o usuário pode postar mensagens e visualizar as mensagens postadas por outros usuários dessa comunidade.

Mural de uma comunidade publica

Na área de membros da comunidade, é disponibilizada uma lista de todos os usuários que são membros da comunidade. Esta lista contém, para cada usuário, a imagem, o nome e o papel do usuário na comunidade, podendo este ser o de usuário comum ou de administrador da comunidade.

Lista de membros de uma comunidade publica

Os Administradores da comunidade podem adicionar membros utilizando o botão "Adicionar Membro". Quando a comunidade é "Restrita" também é exibida a interface de "Requisições de Adesão", onde aparece a lista de usuários que pediram para participar da comunidade. Os administradores da comunidade podem aceitar ou negar o pedido.

Lista de membros de uma comunidade restrita

Ao adicionar uma comunidade, o usuário se torna o único membro e administrador dessa comunidade. Para criá-la, o usuário deve, obrigatoriamente, informar o nome e o nível de privacidade da comunidade que deseja criar e pode, opcionalmente, adicionar uma descrição e uma imagem para essa comunidade.

Interface para adicionar uma comunidade

 

São disponibilizados três níveis de privacidade para as comunidades:

•Público: As comunidades com nível público de privacidade são exibidas na lista de todas as comunidades e qualquer usuário pode participar delas;
•Restrito: As comunidades com nível restrito de privacidade são exibidas na lista de todas as comunidades, mas para participar delas os usuários devem solicitar participação para os administradores da comunidade;
•Privado: As comunidades com nível privado de privacidade não são exibidas na lista de todas as comunidades e os usuários só podem participar delas se forem adicionados diretamente por algum administrador da comunidade.

 

Os administradores de uma comunidade são os únicos que tem permissão para editar ou excluir a comunidade, adicionar usuários à comunidade, remover usuários e alterar o papel dos usuários da comunidade, inclusive dos demais administradores.

Entretanto, as comunidades devem sempre ter ao menos um administrador. Logo, um administrador só pode alterar o seu papel e deixar a comunidade, caso haja outro administrador na comunidade.

As opções de editar e de excluir as comunidades são disponibilizadas aos administradores no cabeçalho das comunidades e as opções de adicionar usuários, excluir usuários ou alterar os papéis dos usuários são disponibilizadas aos administradores, na área de membros das comunidades.
