Para cada interface, e, por sua vez, para cada um de seus sources , podem ser definidos filtros responsáveis pela condição de exibição dos registros.

No exemplo abaixo, para uma determinada interface e para um determinado source, foi definido um filtro utilizando o campo "id". Porém, este filtro não será visualizado pelo usuário (hidden="true").

<sources>

 <source id="subscription" readData="always">

  <fields inherit="all"/>

   <filters>

    <filter id="id" required="true" hidden="true"/>

   </filters>

</source>

</sources>
