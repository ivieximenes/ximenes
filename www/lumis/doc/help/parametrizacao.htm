Parametrização para a interface Menu de Conteúdos

Através da opção Nível do menu acessada através do right-click sobre a interface Menu de Conteúdos, é possível configurar qual nível será o nível foco do menu e quantos níveis abaixo serão exibidos.

Parametrização para a interface Filhos do Conteúdo

Através da opção Nível dos filhos acessada através do right-click sobre a interface Filhos do Conteúdo, é possível configurar quantos níveis abaixo do conteúdo corrente serão exibidos. A estrutura hierárquica está definida no XML para criação de layouts no XSL.
