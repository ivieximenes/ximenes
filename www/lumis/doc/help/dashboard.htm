Dashboard

 Top  Previous  Next

O Dashboard visa facilitar a administração de conteúdos do portal. Ele disponibiliza em um lugar central a administração de todos os conteúdos que o usuário possui permissão de edição.

 

Para acessar o Dashboard, entre na Perfil de Publicador e em seguida selecione o modo Dashboard.

 

Um portal pode ter diferentes seções disponíveis no Dashboard. Pode-se navegar entre as seções utilizando a caixa de seleção conforme demarcado na imagem abaixo:

 

 

As seções disponíveis para seleção depende da montagem da solução (como o serviço de administração automática foi instanciado) e das permissões do usuário atual.

 

A página de Dashboard exibida para os usuários pode ter sido customizado na solução, conforme explicado na montagem do Serviço de Administração Automática. A página de Dashboard padrão possui os elementos explicados abaixo.

 

Meu portal

 

Nesta área é possivel acessar as funcionalidades 'Minhas pendências' e 'Minhas últimas atualizações'.

 

 

Navegação

 

A navegação por “Estruturas de canais” exibirá todos os canais, a partir do canal raiz da seção atual, que possuem serviços com interfaces marcadas como relevantes para auto-administração.

 

A navegação por “Instâncias de serviço” buscará todas as instâncias de serviço que possuam interfaces relevantes para a seção atual.  As interfaces são agrupadas por instância de serviço que são ordenadas por nome.  Torna-se extremamente importante nomear corretamente estas instâncias para que não existam repetições de nomes e para que fique explícita a sua finalidade.

Minhas últimas atualizações

Interface que exibe os últimos conteúdos alterados ou criados pelo usuário.

 

Observações

Na interface minhas últimas atualizações, por questões de performance, não é verificada a permissão de workflow para cada item ao carregar a lista, no entanto as operações que exigem validação fazem esta verificação.

 

 

Minhas pendências

Interface que exibe os conteúdos que têm workflow e não estão nos estados considerados publicado ou arquivado.
