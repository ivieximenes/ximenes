Visualização do Serviço

 Top  Previous  Next

Para o usuário que tenha permissão de visualizar o serviço LikeIt, para cada conteúdo da página que tenha sido selecionado pela query no xsl da interface "LikeIt Scripts", é exibido um link que permite ao usuário demonstrar seu gosto pelo conteúdo.

Ao lado deste link é exibido o total de usuários que gostaram do conteúdo e ao passar o ponteiro do mouse sobre esta exibição do total de usuários, é exibida uma lista ordenada alfabeticamente, contendo o nome de todos os usuários que gostaram desse conteúdo.

Caso o usuário tenha registrado seu gosto pelo conteúdo, o primeiro usuário da lista é exibido como "Você", fazendo referência ao próprio usuário que está visualizando a lista.

Ao clicar no link que registra o gosto do usuário, o link é alterado para permitir ao usuário retirar o registro de seu gosto pelo conteúdo.
