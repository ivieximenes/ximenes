Administração do Conteúdo

 Top  Previous  Next

O serviço de Avisos possui um ambiente administrativo disponibilizado pelo administrador do portal, para o cadastro do conteúdo pelo editor responsável. Para cada conteúdo cadastrado, é possível editá-lo, visualizá-lo e excluí-lo. Para cadastrá-lo, basta acionar “Adicionar” e preencher as seguintes informações:


 

•Título: Título (obrigatório) linkável que compõe a lista de avisos na área administrativa;
•Introdução: Campo obrigatório que contém breve introdução ao assunto;
•Prioridade:Valor atribuído ao conteúdo para efeito de ordenação, filtro e regras de publicação ( prioridade para o destaque entre uma lista de avisos);

Propriedades do Link

•Tipo de link: O conteúdo pode utilizar estrutura de links para um conteúdo interno ou URL;

Caso seja escolhido o valor “Conteúdo Interno”, estará disponível para o cadastro deste conteúdo um editor de texto com formatações. Pode-se também editar o mesmo conteúdo em HTML, selecionando o botão “Código Fonte”.

Caso seja escolhido o valor “URL”, o editor de conteúdo deverá indicar o endereço da página externa (não esquecer o protocolo http:// para páginas externas).

O serviço de Avisos permite que o editor de conteúdo estabeleça regras de publicação como período que determinado conteúdo estará disponível. Permite também publicar o conteúdo para determinados grupos e usuários, para outras instâncias do serviço de Avisos e publicar o conteúdo em outros idiomas.

É possível associar um conteúdo de aviso a outros do mesmo tipo ou referentes a outros serviços (Notícias, FAQ e Eventos). Desta forma, na página de detalhes do aviso, serão exibidos os conteúdos que possuem afinidade de assunto.

Os avisos cadastrados formam uma lista para possíveis manutenções em seus conteúdos. A disponibilização dos campos que aparece na figura abaixo depende de como o administrador do portal configurou o ambiente administrativo de cadastro de conteúdo de avisos.

 

Observação

 

Para informações sobre período de publicação, publicação em outro idioma e visualização de conteúdo, consultar a seção Funcionalidades Gerais dos Serviços de Gestão de Conteúdo. E, sobre como fazer associações entre conteúdos, consultar a seção Associações .
