Datasource

 Top  Previous  Next

É possível configurar o datasource no servidor de aplicação para conexão com o banco de dados. Dessa forma, é delegado ao servidor de aplicação a gerência do pool de conexões.

É necessário que as propriedades de connection.*, dbcp.* and c3p0.* no lumishibernate.cfg.xml sejam removidas ou comentadas  e somente a propriedade abaixo deve ser utilizada:
 

 <property name="connection.datasource">jndiName</property>

 

Para a configuração específica do datasource em cada servidor de aplicação, acesse:

 

Apache Tomcat 8

 

JBoss Enterprise Application Platform 6

 

Oracle Weblogic Server 10.3

 

IBM Websphere AS 7.0
