Geradores de Cache

 Top  Previous  Next

Nesta configuração deve ser associado ao Gerador: um Perfil (que tenha sido previamente criado), filas (que tenham ou não sido criadas anteriormente) e os servidores que vão gerar cache de páginas segundo as configurações do perfil do Gerador.

Os Geradores são gerenciados em: Configuração / Cache HTML / Gerenciador de geradores.

 

A Gestão de Geradores permite as seguintes ações:

•Adicionar Gerador;
•Editar Gerador;
•Remover Gerador.

Adicionar/Editar gerador

Para adicionar um novo Gerador basta selecionar a opção “Adicionar”.

Tela Adicionar Gerador:

As informações necessárias para cadastrar um Gerador são:

•Nome: Nome do Gerador. Campo livre e obrigatório
•Perfil de geração: Perfil a ser utilizado pelo Gerador. Deve ser selecionado um perfil já cadastrado. Campo obrigatório.
•Filas com prioridade: conjunto de filas cujos caches o gerador dará prioridade para gerar.
•Gerar outras filas: Caso esteja desmarcado, os threads do gerador (configuradas no perfil) apenas serão utilizados para gerar cache de páginas pertencentes às filas incluídas no campo "Filas com prioridade". Caso esteja marcado, os threads do gerador poderão gerar cache de páginas de outras filas, caso as filas incluídas no campo "Filas com prioridade" não esteja com alguma geração pendente.
•Servidores: é permitido cadastrar os servidores que vão gerar os cache de páginas segundo as configurações definidas no gerador. Com isso é possível criar um Gerador que utilize servidores que ainda serão adicionados (ou são ativados apenas em determinadas circunstâncias). Para cadastrar todos os servidores basta utilizar o caractere “*”. O número de threads configurado representa o número de threads que executará em cada um dos servidores (não representa o número total).

Excluir gerador

Para excluir um novo Gerador basta selecionar a opção “Excluir”.
