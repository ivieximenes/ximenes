O serviço pode ser montado de diversas maneiras, para o editor de conteúdo e para o usuário final da solução,dependendo de como foi arquitetado pelo administrador do portal.

Para utilizar o serviço Perfil Social, basta instanciá-lo em uma canal e arrastar suas interfaces para as áreas das páginas em modo de edição. Suas interfaces correspondem a um ambiente de atualização do conteúdo e áreas de visualização do serviço pelo usuário final.

 

•Atualização: Esta interface permite ao usuário a manutenção das informações do seu perfil social;
•Perfil público: Esta interface disponibiliza a uma visualização do dos dados públicos de um perfil social, para o usuário final da solução.
•Perfil Completo: Esta interface disponibiliza a uma visualização do dos dados completos de um perfil social, para o usuário final da solução.

 

Observação

O Lumis Portal oferece como sugestão de montagem deste serviço um prebuilt channel com uma estrutura de canal, páginas, e disposição das interfaces nas páginas.

Para informações sobre como instanciar canais e páginas; consultar a seção sobre Portal Studio .

Através do right-click da interface, encontram-se disponíveis parametrizações para a instância de serviço e customizações para a interface. Consultar a seção Customização de instâncias de Interfaces.
