Controle de acesso do Portal

Gerenciar segurança do portal

(TODAS AS PERMISSÕES DO PORTAL)

Permite gerenciar o controle de acesso ao Lumis Portal.

Porém, além de um usuário possuir todas as permissoes  para gerenciar o portal, ele deve ser membro do grupo "Administradores do Portal" para poder gerenciar as outras áreas funcionais do produto:  "Configurações".

Pelo menos um usuário deve possuir esta permissão.
