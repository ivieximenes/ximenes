Web Services

 Top  Previous  Next

Os Web Services oferecem acesso a métodos (funcionalidades) de alguma biblioteca ou API, através de chamadas HTTP, utilizando XML e SOAP. No caso específico do Lumis Portal, é possível, a partir de qualquer plataforma e qualquer linguagem de programação que tenha suporte a web services, utilizar funcionalidades da API do produto.

A principal função é facilitar a interoperabilidade entre o Lumis Portal e sistemas desenvolvidos em outras plataformas, dando-lhes acesso, por exemplo, a informações de usuários cadastrados no Lumis, e a funcionalidades do portal, como autenticação e criação de usuários.

A utilização de Web Services faz do Lumis Portal o elemento de integração entre diversas aplicações e plataformas de uma empresa.
