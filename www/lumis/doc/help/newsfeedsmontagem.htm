Montagem

Esta seção refere-se à montagem do serviço pelo administrador do portal, para o usuário final da solução. Não há uma montagem do ambiente administrativo de conteúdo para o editor de conteúdo.

Para utilizar o serviço NewsFeeds, basta instanciá-lo em um canal e arrastar sua única interface para a página onde se deseja permitir a postagem de conteúdos a serem compartilhados entre os usuários.

·NewsFeed: Esta interface uma caixa de texto onde o usuário pode postar conteúdos e uma lista com paginação, contendo os últimos conteúdos postados.

 

 

 

Observação

 

Para informações sobre como instanciar canais, páginas, e atribuição de permissões no controle de acesso, consultar a seção sobre Portal Studio .

Através do right-click da interface, encontram-se disponíveis parametrizações para a instância de serviço e customizações para a interface. Consultar a seção Customização de instâncias de Interfaces.
