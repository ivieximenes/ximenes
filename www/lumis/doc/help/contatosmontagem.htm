Montagem

O serviço pode ser montado de diversas maneiras, para o editor de conteúdo e para o usuário final da solução,dependendo de como foi arquitetado pelo administrador do portal.

Para utilizar o serviço Contatos, basta instanciá-lo em um canal e arrastar suas interfaces para as áreas das páginas em modo de edição. Suas interfaces correspondem a um ambiente administrativo de cadastro do conteúdo e áreas de visualização do serviço pelo usuário final.

 

•Administração: Esta interface permite o cadastro e manutenção das informações dos contatos pelo editor de conteúdo;
•Lista: Esta interface disponibiliza a lista de contatos cadastrados, para o usuário final da solução. Por default, são exibidos todos os contatos cadastrados. Através da lista alfabética pode-se filtrar os contatos para melhor localizá-los. Também pode-se usar o campo "Busca" para localizar um contato, informando parte (utilizar *) ou toda a palavra correspondente a uma informação do contato desejado. Desta forma, serão exibidos os contatos  que atendem total ou parcialmente a busca.
•Detalhes: Esta interface exibe as informações do contato selecionado a partir da interface Lista e Administração.

Para disponibilizar o serviço Contatos para o usuário final, basta instanciar as interfaces Lista e Detalhes em páginas diferentes do canal. A página que contém a interface Detalhes deve ter a propriedade “oculto” selecionada, pois esta página será acessível a partir das páginas correspondentes às interfaces Administração e Lista .

O ambiente de administração de contatos requer apenas uma página em um canal restrito do portal, onde apenas os editores de conteúdo terão acesso, após o login. A fim de proporcionar uma maior organização no portal, é aconselhável que esta página pertença a um canal específico, o qual está sendo chamado aqui de “Contatos”.

Atribuição de Permissões

Para que o editor de conteúdo possa acessar o ambiente administrativo de contatos, o administrador do portal deve atribuir a ele a permissão “Editar conteúdo de instância de serviço”. Esta permissão é concedida no controle de acesso da instância do serviço.

Observação

 

O Lumis Portal oferece como sugestão de montagem deste serviço um prebuilt channel com uma estrutura de canal, páginas, e disposição das interfaces nas páginas.

Para informações sobre como instanciar canais, páginas, e atribuição de permissões no controle de acesso, consultar a seção sobre Portal Studio .

Através do right-click da interface, encontram-se disponíveis parametrizações para a instância de serviço e customizações para a interface. Consultar a seção Customização de instâncias de Interfaces.
