Para utilizar o serviço Gerenciador de Usuários , basta instanciá-lo em um canal e arrastar a sua interface Administração em uma página do portal. Esta interface permite o cadastro e manutenção de usuários no portal.

Parametrização da Instância de Serviço

Pode-se configurar para qual domínio os usuários serão cadastrados. Para isso, o administrador deve selecionar via right-click qualquer interface do serviço de gerenciador de usuários e informar o domínio.

 

Ao selecionar a opção domínio, é disponibilizada uma lista de domínios, um deles o domínio "global" e os outros (se houver), domínios locais representados pelos prefixos dos canais que possuem usuários locais.

 

Se o administrador determinar que o domínio será "global", o cadastro será de usuários considerados globais ao portal. Caso o administrador escolha algum domínio local, o cadastro de usuários será apenas para determinado canal representado por aquele domínio.

 

 

Se o administrador determinar que o domínio será "Todos", o cadastro será de usuários considerados globais ao portal, porém, a interface administrativa exibirá tanto usuários globais quanto usuários locais.

Atribuição de Permissões

Para que um usuário possa cadastrar usuários globais, ele deve ter a permissão "Gerenciar Grupo" no controle de acesso global. Este controle de acesso é aquele acessado via right-click sobre qualquer canal, na opção "Gerenciar Acesso do Portal".
