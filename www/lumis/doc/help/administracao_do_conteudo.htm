Administração do Conteúdo

 Top  Previous  Next

O serviço de Questionários possui um ambiente administrativo para questionários, para questões e para lista de respostas, disponibilizado pelo administrador do portal para o editor de conteúdo. O conteúdo cadastrado pode ser alterado, visualizado ou excluído.

 

Administração de Questionários

Esta interface permite a criação, alteração e remoção de questionários. Ao remover um questionário, todas as suas perguntas e respostas serão apagadas.

Além disso, através do botão "Baixar respostas" é possível exportar um arquivo CSV contendo todas as respostas de um determinado questionário codificado em UTF-8 e separado por ;. Também é possível acessar a administração de respostas através do botão "Lista de Respostas".

Adicionar Questionário

Para cadastrar questionários, basta acionar “Adicionar” no ambiente administrativo e preencher as seguintes informações:


 

•Título: Título (obrigatório) do questionário;
•Permitir responder mais de uma vez: Opção que permite ou não que usuário responda o questionário mais de uma vez. É importante notar que essa validação só ocorre para usuários autenticados no portal, ou seja, caso o usuário seja Guest essa validação não ocorrerá;
•Respostas anônimas: Opção que torna as respostas dos usuários, em repostas anônimas;

O serviço de Questionários permite que o editor de conteúdo estabeleça regras para publicação como período que determinado conteúdo estará disponível para o usuário final da solução. Permite também publicar o conteúdo para determinados grupos e usuários.

Por ser o serviço Questionários um serviço de gestão de conteúdos, permite a utilização de workflow para aprovação dos mesmos.

Os questionários cadastrados formam um conjunto de registros para posteriores manutenções em seus conteúdos. A disponibilização dos campos que aparece na figura abaixo depende de como o administrador do portal configurou o ambiente administrativo de cadastro de questionários.

Administração de Questões

A partir de um questionário cadastrado, é que são cadastradas as questões. Para isso as seguintes informações devem ser preenchidas:

•Título: Título (obrigatório) para a questão;
•Tipo: Tipo (obrigatório) para a resposta a questão, podendo ser:
1.Resposta múltipla - dada uma lista de opções, o usuário tem a opção de selecionar quantas respostas achar plausível.
2.Resposta individual - dada uma lista de opções o usuário tem a opção de selecionar um e somente uma resposta.
3.Combo de seleção -  dada uma lista de seleção, o usuário tem a opção de selecionar um e somente uma resposta.
4.Texto - resposta discursiva com campo de texto limitada em XX caracteres.
5.Texto curto - resposta discursiva com campo de texto limitada em XX caracteres.
6.Verdadeiro ou Falso - opção de verdadeiro ou falso como resposta.
7.Número - resposta em campo de texto limitada para aceitar apenas números.
8.Data - campo de texto limitado a aceitar apenas data .
•Número de opções: permite determinar o número de opções de resposta que serão cadastradas, este campo será mostrado apenas para questões do tipo; Resposta múltipla, Resposta individual, Combo de seleção. É uma lista contendo o número de opções que serão disponibilizadas como resposta (2 à 10 opções);
•Opção n: (obrigatório quando visualizável) permite cadastrar uma das opções de resposta, sendo que a quantidade deste campo está relacionada a opção selecionada no campo Número de opções;
•Ordem da questão: (obrigatório) valor numérico da posição desta questão em seu questionário. A administração permite que duas perguntas sejam cadastradas com o mesmo valor desse campo. Nesse caso, a ordem entre essas perguntas não será determinada;
•Questão obrigatória: opção de torna a questão ser obrigatória ou não;

As questões cadastradas formam um conjunto de registros para posteriores manutenções em suas informações. Para editar as informações de uma questão ou mesma excluí-la, basta selecioná-la e acionar as opções desejadas.

Lista de Respostas

Esta inteface lista todas as respostas efetuadas para um determinado questionário.

 

•Data: Data e hora em que a resposta foi efetuada pelo usuário;
•Nome Completo: Nome Completo do usuário que preencheu a resposta. Este campo só aparece caso o questionário não seja anônimo;

Através da lista de respostas, é possível visualizar as respostas de um determinado usuário.

Baixar Respostas

Através deste botão, é possível exportar um arquivo CSV contendo todas as respostas de um determinado questionário codificado em UTF-8 e separado por ;.

Ver Respostas

Interface que descreve as respostas do cliente junto com suas perguntas.

Obs: Não é permitido alterar os dados das respostas de um usuário.

 

Observação
