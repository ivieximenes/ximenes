Visualização do Serviço

 Top  Previous  Next

O usuário final do portal pode interagir das seguintes maneiras com o serviço de Login.

•Tela de Logout: Exibe mensagem para o usuário logado e um link  para realizar o logout do portal;

 

•Tela de Login : Autenticação do usuário na solução no portal. A autenticação é requerida também, quando o usuário tentar acessar uma página que não possui acesso.

 

•Tela de Atualização do cadastro do usuário:  

Observação

 

É necessário que o usuário cadastrado tenha a permissão de visualização da instância de serviço para poder logar no portal.
