Administração do Serviço

Uma barra de navegação é composta de itens de menus que são cadastrados em uma área administrativa pelo próprio administrador do portal. Para cadastrar cada item de menu, deve-se acionar “Adicionar” na área administrativa e preencher as seguintes informações:

• Nome: Nome do item de menu linkável para uma página ou URL;

• Descrição: Breve explicação do item de menu cadastrado;

•Conteúdo Pai: Permite informar qual será o item de menu “pai” (já cadastrado) do item a ser cadastrado. Quando o item de menu "pai" estiver no nível zero, somente será exibido caso a propriedade “Esconder o nível zero" habilitada;

•Imagem: Campo facultativo que exibe a imagem cadastrada no repositório de imagens escolhido . Os XSLs padrão do produto não renderizam a imagem cadastrada;

•Posição: Campo obrigatório que determina a posição do item de menu na barra de navegação;

•Abrir em: Pode-se determinar se a página que o item de menu referencia, abrirá na mesma janela, em uma nova janela ou em um pop-up;

• Tipo: Campo obrigatório que determina se o item de menu assumirá a forma de link para uma URL, árvore de canais (para escolher uma estrutura hierárquica de canais e páginas para representar) ou se a forma estará determinada na implementação de uma classe java;

Caso o item de menu assuma forma de link, é necessário informar a URL (com o protocolo http://) que ele referenciará:

Caso o item de menu seja do tipo Árvore de Canais, é necessário informar uma  estrutura de canais do portal que irá compor o menu. O nome atribuído ao item de menu substituirá o canal pai da estrutura escolhida. Em seguida, informar o número de níveis da estrutura que irão compor o menu (escolha de 0 a 10 níveis, ou valor ilimitado). Por fim, é possível determinar se o item do menu (correspondente ao nível "0") será exibido ou não na barra de navegação.

Caso o item de menu assuma o tipo implementado em uma classe, é necessário informar o nome de tal arquivo.

Os itens de menu cadastrados formam uma lista no ambiente administrativo para possíveis manutenções em seus cadastros (atualizações ou exclusões).
