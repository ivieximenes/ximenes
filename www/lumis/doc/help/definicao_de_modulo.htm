Definição de Módulo

 Top  Previous  Next

A definição de um módulo é realizada através de um arquivo de marcação no formato XML, que deve seguir o arquivo de esquema disponibilizado pelo portal. O link para esse arquivo de esquema pode ser encontrado na página API e XSD.

 

A definição do módulo é obrigatória e deve ser realizada com um arquivo chamado lumis-module.xml. Esse arquivo deve estar localizado em uma pasta chamada META-INF. Essa pasta, por sua vez, deve estar localizada na raiz do módulo.

 

Nesse arquivo podem ser definidos:

•Identificador do módulo: Esse campo é obrigatório e fornece o identificador do módulo.
•Nome de exibição: Opcional. Fornece o nome de exibição para esse módulo.
•Descrição: Opcional. Fornece uma descrição para o módulo.
•Versão: Opcional. Indica a versão do módulo.

Exemplo de descritor de módulo

<?xml version="1.0" encoding="UTF-8"?>

<lumis-module xmlns="http://www.lumis.com.br/lumisportal/xsd/module">

 <id>br.com.empresa.modulo1</id>

 <version>1.0</version>

 <display-name>empresa.modulo1</display-name>

</lumis-module>
