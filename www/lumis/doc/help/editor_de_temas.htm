Editor de temas

O editor de temas permite visualizar e editar os arquivos de um tema no próprio navegador. Para abrir o editor de tema basta clicar no link "Editar arquivos" na administração de temas.

 

O editor é dividido em 3 areas:

1.Lista de arquivos: É exibida a estrutura da pasta do tema.
2.Visualização de arquivos: Exibe o conteúdo dos arquivos e permite a edição de arquivos em texto. Imagens não podem ser editadas.
3.Botões de ação: Permite salvar o arquivo sendo exibido ou todos abertos para edição, fazer upload de multiplos arquivos compactados em formato .zip e fechar o editor.

 

Menus de contexto: O editor oferece diversos menus de contexto, na lista de arquivos, nos diretorios, nos arquivos e nas abas com o titulo dos arquivos.

1.O menu de contexto da lista permite criar os diretorios www e def, fazer upload de arquivos em formato zip, baixar o tema em formato zip e atualizar a listagem com modificações externas.

               
 

2.O menu de contexto dos diretórios permite criar novos arquivos, criar novas pastas, fazer upload de arquivos e arquivos em formato zip, baixar o diretório em formato zip, renomear, excluir ou atualizar o diretório.

               
 

3.O menu de contexto dos arquivos permite renomear e excluir o arquivo.

 
4.O menu de contexto das abas permite fechar a aba ou fechar todas as outras abas exceto a aba selecionada.

 

Para maiores informações sobre temas, veja Temas do Portal.
