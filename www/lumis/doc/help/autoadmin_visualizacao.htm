Este serviço visa facilitar o gerenciamento dos conteúdos do portal e deve ser acessado pelo item 'Dashboard' localizado no menu superior na visão do publicador.

 

Meu portal

 

Neste área é possivel acessar as funcionalidades 'Minhas pendências' e 'Minhas últimas atualizações'.

 

 

Navegação

 

A navegação por “Estruturas de canais” exibirá todos os canais, a partir do canal raiz da auto-administração, que possuem serviços com interfaces marcadas como relevantes para auto-administração.

 

A navegação por “Instâncias de serviço” buscará todas as instâncias de serviço que possuam interfaces relevantes para a auto-administração do canal raiz relativo.  As instâncias de interface são agrupadas por instância de serviço que são ordenadas por nome.  Torna-se extremamente importante nomear corretamente estas instâncias para que não existam repetições de nomes e para que fique explícita a sua finalidade.

Minhas últimas atualizações

Interface que exibe os conteúdos alterados ou criados pelo usuário.

 

Observações

Na interface minhas últimas atualizações, por questões de performance, não é verificada a permissão de workflow para cada item ao carregar a lista, no entanto as operações que exigem validação fazem esta verificação.

 

Minhas pendências

Interface que exibe os conteúdos que têm workflow e não estão nos estados considerados publicado ou arquivado.
