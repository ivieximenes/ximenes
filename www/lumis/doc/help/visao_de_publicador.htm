Perfil de Publicador

 Top  Previous  Next

Este perfil é similar ao perfil de Usuário, porém, nele é possível efetuar tarefas administrativas através dos modos pertencentes à este perfil, como por exemplo, a Administração de Conteúdo.

O acesso à este perfil é possível através do dropdown disponibilizado ao canto superior esquerdo da página, clicando em um dos modos destacados a seguir:

 

 

O perfil de Publicador engloba os modos, Administração de conteúdo, Edição em contexto e Posicionar widgets.

 

Permissionamento:

 

Para um usuário ter acesso ao perfil de Publicador, é necessário o mesmo ter a permissão em ao menos um de seus modos. Para permitir que um dado usuário acesse o perfil, é necessário acessar o canal ou página desejada, clicar com o botão direito do mouse e então "Controle de Acesso". No pop exibido, selecionar o usuário desejado e marcar o checkbox "Publicar Conteúdo", que daria permissão a Personalização por exemplo. Caso desejado negar o acesso ao modo, é necessário selecionar o checkbox "Negar". Para maiores informações sobre Controle de Acesso, visite Controle de Acesso do Portal.

 

 

Em caso de um usuário não possuir permissões suficientes ou não possuir permissão em nenhum modo do perfil de Publicador, os botões não aparecerão.

 

Navegação entre Modos:

De cada um destes modos é possível navegar para os demais, desde que o usuário possua tal permissão, em caso negativo, a opção pelo modo estará desabilitada. Para navegar ao Modo desejado, é necessário apenas clicar sobre o modo na barra exibida.

 

Navegação entre Visões:

A partir deste perfil, é possível navegar para o perfil de Administrador (Portal Studio, Configurações) se estiver permitido, através do dropdown de modos disponibilizados e para o perfil de Usuário (Navegar como usuário), que além de ser acessível pelo dropdown de modos, é acessível também pressionando a tecla F2.

 

 

 

Observações

 
•Para informações detalhadas sobre os modos do perfil de Publicador, por favor, visite a seção dos modos de Publicador.

 

•Não é gerado cache nesta perfil.
