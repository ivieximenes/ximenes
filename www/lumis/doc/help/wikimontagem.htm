O serviço pode ser montado de diversas maneiras, para o administrador do serviço e para o usuário final da solução, dependendo de como foi arquitetado pelo administrador do portal.

Para utilizar o serviço Wiki, basta instanciá-lo em um canal e arrastar suas interfaces para as áreas das páginas em modo de edição. Suas interfaces correspondem a um ambiente administrativo de exclusão do artigo e áreas de visualização, criação, edição entre outras ações realizadas pelo usuário final.

 

•Adicionar Artigo: Esta interface permite criar um artigo. É acessada a partir do botão Criar disponível na interface Exibir Artigo;

 

•Administração de Artigos: Esta interface permite excluir um ou mais artigos pelo administrador do serviço. É acessada a partir da interface Exibir Artigo;
 
•Histórico do Artigo: Esta interface exibe todas as versões do artigo corrente com a possibilidade de editar uma delas. É exibida a partir da seleção do botão com o mesmo nome;
 
•Editar Artigo: Esta interface que permite editar o artigo corrente. É exibida a partir da seleção do botão Editar disponível na interface Exibir Artigo;
 
•Editar Artigo pela Versão: Esta interface permite editar uma versão antiga ou nova criando uma outra versão que estará disponível para visualização. É exibida a partir da seleção do botão Editar disponível na interface Histórico do Artigo;
 
•Exibir Artigo: Interface principal, cuja página deve ser a primeira a ser acessada pelo usuário final. Exibe o artigo corrente, acessada pelo resultado de busca ou por um item de menu;
 
•Artigos desejados: Esta interface permite criar novos artigos, seja através da edição de um artigo sem conteúdo ou criando um novo. É acessada a partir da interface Exibir Artigo.

Para disponibilizar o serviço Wiki para o usuário final, basta instanciar as interfaces Adicionar Artigo, Histórico do Artigo, Editar Artigo, Editar Artigo pela Versão, Exibir Artigo, e Artigo Desejadose Lista Rápida em páginas diferentes do canal. A página que contém a interface Histórico do Artigo deve deve ter a propriedade “Oculto” selecionada, pois esta página será acessível a partir da página de exibição do artigo corrente. As páginas que contem as interfaces Adicionar Artigo, Editar Artigo, Editar Artigo pela Versão e Artigos Desejados também devem ter a propriedade "Oculto" selecionada pois são acessadas a partir de botões principalmente da interface Exibir Artigo.

O ambiente administrativo do serviço requer apenas uma página em um canal restrito do portal, onde apenas os editores de conteúdo terão acesso, após o login. A fim de proporcionar uma maior organização no portal, é aconselhável que esta página pertença a um canal específico, como por exemplo, “Wiki”. Por outro lado, pode também configurar a página administrativa com a propriedade "Oculto" já que ela também pode ser acessar a partir do botão Administração de Artigos.

 

Atribuição de Permissões

O serviço Wiki trabalha com três perfis de utilização do serviço:

•Editar conteúdo de instância de serviço: Usuário com esta permissão criar artigos com ou sem conteúdos e relacioná-los com outros. Na interface principal do Wiki são exibidas as opções Editar, Histórico do Artigo, Criar e Artigos Desejados;
 
•Gerenciar instância de serviço: Usuário com esta permissão pode, além de visualizar e criar artigos, excluí-los quando necessário. Na interface principal do Wiki além das opções acima citadas, é exibida a opção Administração de artigos.

Estas permissões são concedidas no controle de acesso da instância do serviço Wiki.

Na figura a seguir, há um exemplo do controle de acesso da instância de serviço utilizada, cujo usuário "Wiki 3" tem a permissão de "Gerenciar instância de serviço" e consequentemente tem a permissão "Editar Conteúdo".

Para que o editor de conteúdo realize upload de arquivos, é necessário que ele tenha a permissão de Efetuar Upload de arquivo no Gerenciador de acesso do portal.
 

Observação

 

O Lumis Portal oferece como sugestão de montagem deste serviço um prebuilt channel com uma estrutura de canal, páginas, e disposição das interfaces nas páginas.

Para informações sobre como instanciar canais, páginas e atribuição de permissões no controle de acesso, consultar a seção sobre Portal Studio .

Através do right-click da interface, encontram-se disponíveis parametrizações para a instância de serviço e customizações para a interface. Consultar a seção Customização de instâncias de Interfaces. Entre elas o repositório de imagens a ser utilizado no editor HTML.
