Montagem do Serviço de Login

 Top  Previous  Next

O serviço pode ser montado de diversas maneiras, dependendo de como foi arquitetado pelo administrador do portal.

Para utilizar o serviço de Login, basta instanciá-lo em um canal e arrastar suas interfaces para as áreas das páginas em modo de edição.

•Login : Esta interface permite a autenticação do usuário da solução;
•Atualizar Cadastro: Esta interface permite que o usuário final altere seus dados. Pode ser utilizado o serviço de Links para remeter o usuário para a página que contenha esta interface;
•Logout: Esta interface permite que o usuário desconecte do portal.
 

Observação

 

Para informações sobre como instanciar canais e páginas, consultar a seção sobre Portal Studio .

Através do right-click da interface, encontram-se disponíveis parametrizações para a instância de serviço e customizações para a interface. Consultar a seção Customização de instâncias de Interfaces.
