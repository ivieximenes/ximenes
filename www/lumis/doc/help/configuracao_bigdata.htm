Através da administração de Big Data é possível reindexar os conteúdos dos serviços que são buscáveis.

 

Esta interface possui três botões:

Reindexação de Serviço

Ao selecionar ao menos um serviço da lista e clicar no botão "Reindexar Serviços", a seguinte tela irá aparecer:

Tela de progresso, onde é possível visualizar quais serviços já foram reindexados e quais ainda precisam ser feitos. Note que, se o processo for interrompido prematuramente, apenas as interfaces com o estado "Finalizado" serão reindexadas.

Nas reindexações de serviço, as estruturas BigData relativas a todos os Sources daquele serviço são reconstruídas.

Sources do tipo "table" serão processados apenas neste tipo de reindexação, isto ocorre porque os dados destes Sources são considerados globais e, portanto, não pertencentes a uma instância de serviço. Já Sources do tipo "contentTable" serão reindexados em ambos os tipos de reindexação.

 

Observações

 
•Quando um serviço for registrado e as alterações não puderem ser refletidas de forma que não conflite com a estrutura anteriormente armazenada no repositório de BigData, este serviço deve ser reindexado, através dessa página, para que sua estrutura BigData seja reconstruída de acordo.

Reindexação de Instância de Serviço

Ao selecionar ao menos um serviço e clicar em "Reindexar Instâncias de Serviços" a seguinte tela irá aparecer:

Nesta tela, o usuário poderá escolher uma ou mais instâncias daqueles serviços previamente selecionados para reindexar.

 

Após selecionar o botão "Reindexar" o usuário será enviado a uma tela de progresso similar á tela de serviço, só que listando as instâncias de serviço selecionadas.

O Lumis Portal adiciona, atualiza e exclúi automáticamente os documentos indexados relativos aos conteúdos quando esses são adicionados, editados ou excluídos, respectivamente.

Nas reindexações de instância de serviço o Lumis Portal não reconstrói as estruturas BigData relacionadas aos Sources dos serviços. Ele apenas elimina os documentos armazenados pertencentes à instância de serviço e tenta recriá-los utilizando os dados atuais.

 

Reindexar todos os Serviços

Ao selecionar este botão, o portal irá lhe enviar diretamente para esta tela:

Nesta mesma, será mostrada uma lista contendo todos os serviços que permitem reindexação, listados alfabeticamente, é importante lembrar que se esta interface for fechada prematuramente, apenas os serviços onde o estado esteja como "Finalizado", serão reindexados.
