Cada canal poderá ter somente uma instância do serviço de auto-administração.  Isso, por sua vez, não impede que os sub-canais contenham suas próprias instâncias de auto-administração.

 

Observação

Apenas usuários com permissão para editar conteúdo na instância do serviço de auto-administração são capazes de acessar o Dashboard para essa instância do serviço.
As permissões para uma instância de serviço podem ser configuradas acessando o Controle de Acesso da mesma através da Administração de Instâncias de Serviços.

 

Customização

O Lumis Portal provê uma página padrão de Dashboard.  Não existe acesso algum à esta página senão pela barra de modos do publicador.  Esta página possui permissões especiais e por isso não é editável por nenhum usuário (incluindo administrador do portal). Existe, contudo, uma forma de customizar uma página de auto-administração.

 

Para criar uma página customizada de Dashboard, é necessário criar uma página e adicionar as interfaces de Navegação e Holder.

 

Observação

Criar uma página que não contenha uma interface de Holder pode impedir o acesso ao Dashboard, não habilitando o botão na barra da visão do Publicador.

 

 

Depois é necessário indicar na instância da auto-administração qual página será usada.  Para informar, vá a instância do serviço e edite-o.  Na aba “Auto Administração”, você poderá escolher a página.

 

 

Um canal que possui uma instância deste serviço é considerado raiz de uma seção do Dashboard. Desta forma, ao entrar no Dashboard, o portal identificará esta raiz para exibir as informações da árvore de canais/instâncias de serviço correspondente.

 

Logo, com isso é possível definir diferentes páginas de Dashboard para um portal, bastando para cada página desejada, montar a página correspondente no portal e associá-la a uma instância de serviço de Administração Automática. Com isto os usuários visualizarão no modo Dashboard as seções correspondentes às instâncias de serviço de Administração Automática a que eles têm acesso.
