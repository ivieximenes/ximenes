Versão 5.0.1

Melhorias:

 

•Foram feitas três melhorias de performance:
oOtimização em lumis.util.XmlUtil para reduzir contenção durante processamentos paralelos. Melhoria já incluida na versão 4.2.2.091119;
oAlteração do cache de usuários em UserDaoJdbc para usar fetch para leitura;
oMudança na classe CookieUtil, permitindo que as propriedades do usuário Guest sejam lidas de forma estática.
•Possibilidade de configuração do número de posts exibidos por página do blog.

 

Defeitos:

 

•Corrigido defeito ao gerar o arquivo dinâmico Robots.txt quando a URL não é configurada. Para mais detalhes, consulte a configuração da URL Robots.txt;
•Corrigido defeito dos parâmetros da página que não estavam sendo decodificados;
•Corrido defeito com as URLs amigáveis no ambiente com o Weblogic 10.0;
•Corrigido defeito na limpeza de cache de ChannelWebResource;
•Corrigido defeito do código responsável por exibir o banner em flash, que era carregado dinamicamente com o uso de Javascript após o carregamento inicial da página;
•Corrigido defeito os arquivos de cache HTML são apagados;
•Corrigido defeito no método fixReferencesAccordingToPath que era carregado dinamicamente com o uso de Javascript;
•Corrigido defeito com a referência de imagens inseridas no editor HTML com URLs amigáveis habilitadas, ao reparar os paths dos códigos HTML gerados pelo Lumis as referências das URLs do FCK Editor se tornavam inválidas;
•Corrigido defeito em que o serviço Wiki permitia criar um link para artigo novo com um título já existente;
•Corrigido defeito no filtro da interface de seleção de artigo (link para artigo) do serviço Wiki;
•Corrigido o defeito em que a Querystring era incluida erroneamente em WebServerIntegrationForwardFilter;
•Corrigido o erro de ordenação na lista de post no serviço Blog;
