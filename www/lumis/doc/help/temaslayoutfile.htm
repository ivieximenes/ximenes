Substituição de arquivo de layout

 Top  Previous  Next

Os Arquivos de layout de um tema, devem ser incluídos dentro de uma pasta def filha da pasta onde está o themedefinition.xml correspondente. Dentro deste def deve ser criado o caminho de pastas de igual ao do Layout File que se deseja substituir.

 

Dessa forma, quando uma página for ser renderizada, caso um tema aplicado nesta página possua um Layout File com mesmo caminho do que está configurado na página, o arquivo incluído no tema será utilizado ao invés do original.
