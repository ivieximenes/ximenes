Versão 6.1.0

 Top  Previous  Next

A versão 6.1.0 possui todas as modificações incluídas até a versão 6.0.2. Abaixo seguem os destaques das demais funcionalidades não existentes na versão 6.0.2.

 

Novidades

 

•Incluído o controle que permite upload de múltiplos arquivos;

 

•Incluídos serviços de rede social do Lumis Portal: Rede Social, Perfil Social e Microblog;

 

•Incluído o serviço de integração com redes sociais públicas: Twitter e Facebook;

 

•Incluído suporte a operações REST em serviços.  Serviços DOUI possuem nativamente operação para leitura em sources;

 

•Incluída a funcionalidade de transformações automáticas de arquivos inseridos no portal.

 

 

Melhorias

 

•Filtros de post-load agora dão suporte aos operadores lessThanOrEqual e greaterThanOrEqual;

 

•Adicionada a ação de expirar cache no Gerenciador de Cache;

 

•Property bags podem ser definidas para instâncias de serviço;

 

•Utilização do controle de upload de múltiplos arquivo para os serviços de Álbum de Fotos, Documentos e Imagens.

 

 

Defeitos

 

•Filtro e paginação no pop de selecionar termo pai não funcionava no serviço de Categorização.

 

 

Alterações

 

•Os seguintes servidores de aplicação pararam de ser suportados:

 

Websphere Application Server 6.1

 

•Os seguintes servidores de aplicação começaram a serem suportados:

 

Websphere Application Server 7.0

 

•As seguintes bibliotecas foram adicionadas ao WEB-INF/lib:

 

jersey-client-1.8.jar

jersey-core-1.8.jar

jersey-json-1.8.jar

jersey-server-1.8.jar

json.jar

restfb-1.6.6.jar

twitter4j-core-2.2.4.jar

 

•As seguintes bibliotecas foram removidas do WEB-INF/lib:

 

jtidyservlet-r8-SNAPSHOT.jar
