O idioma do portal pode ser alterado pelo usuário final, ao selecionar uma das opções de localizações regionais disponíveis, na interface Selecionador de Localização, cadastradas pelo administrador do serviço.

A figura acima é um exemplo da interface Selecionador de Localização cujo nome, para este exemplo, foi customizado para "Idioma".

O portal exibirá em listas e resultados de busca apenas os conteúdos e objetos para determinado idioma. Os conteúdos default são exibidos independente do idiomar da sessão do usuário.
 

Observação

 

É necessário que o usuário tenha a permissão de visualização da instância de serviço para poder visualizar a lista de termos.
