*********************************
*  LUMIS PORTAL 8.1.1.150831    *
*********************************

O Lumis Portal é um conjunto de softwares baseados em tecnologia 
Java(TM) para criação, gerenciamento e integração de portais.

Nesta distribuição encontram-se os arquivos necessários para instalar, 
configurar e utilizar o Lumis Portal 8.1.1.150831.

Estrutura
---------
 |__lib - bibliotecas compartilhadas.
 |__lptf - framework de testes do Lumis Portal.
 |__lumisdata - arquivos de configuração do portal.
 |__setup - arquivos para setup do banco.
 |__www - aplicação Web.

Instalação
----------
 Para maiores informações sobre a instalação e configuração do Lumis Portal 
 leia a documentação em html disponível no manual em html da documentação do produto.
 
 Após a instalação, utilize o login admin com a senha em branco, para autenticar 
 a primeira vez no portal.

Observações: 
 O Microsoft Internet Explorer 8 ou superior é requerido para utilização do 
 portal no ambiente administrativo.
